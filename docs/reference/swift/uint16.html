init(bitPattern: Int16)

Construct a UInt16 having the same memory representation as the Int16 bitPattern. No range or overflow checking occurs, and the resulting UInt16 may not have the same numeric value as bitPattern–it is only guaranteed to use the same pattern of bits.
UInt16をInt16 bitPatternと同じメモリ表現を持つようにして組み立てます。範囲またはオーバーフロー検査が起こることはありません、そして結果のUInt16はbitPatternと同じ数値を持たないでしょう - それはただ同じビットのパターンを使うことだけを保証されます。
