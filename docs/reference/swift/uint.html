UInt

An unsigned integer value type.
符号なし整数値型。

Language

SDK

Overview
概要

On 32-bit platforms, UInt is the same size as UInt32, and on 64-bit platforms, UInt is the same size as UInt64.
32ビットプラットホーム上では、UIntはUInt32と同じ大きさです、そして64ビットプラットホーム上では、UIntはUInt64と同じ大きさです。

init(bitPattern: Int)

Construct a UInt having the same memory representation as the Int bitPattern. No range or overflow checking occurs, and the resulting UInt may not have the same numeric value as bitPattern–it is only guaranteed to use the same pattern of bits.
UIntをInt bitPatternと同じメモリ表現を持つようにして組み立てます。範囲またはオーバーフロー検査が起こることはありません、そして結果のUIntはbitPatternと同じ数値を持たないでしょう – それはただ同じビットのパターンを使うことだけを保証されます。
