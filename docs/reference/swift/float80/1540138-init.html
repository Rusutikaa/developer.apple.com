Parameters
パラメータ

text

The input string to convert to a Float80 instance. If text has invalid characters or is in an invalid format, the result is nil.
Float80インスタンスへ変換することになる入力文字列。textが無効な文字を持つまたは無効な形式であるならば、結果はnilです。

Discussion
解説

The string passed as text can represent a real number in decimal or hexadecimal format or special floating-point values for infinty and NaN (“not a number”).
textとして渡される文字列は、10進または16進数形式における実数、または無限大とNaN（「非数」）に対する特別な浮動小数点値を表すことができます。

The given string may begin with a plus or minus sign character (+ or -). The allowed formats for each of these representations is then as follows:
与えられた文字列は、プラスまたはマイナス符号文字（+または-）で始まっても構いません。これらの表現のそれぞれに対する許される形式はそれで以下になります：

  • A decimal value contains the significand, a sequence of decimal digits that may include a decimal point.
    10進の値は仮数、小数点を含んでも良い一連の10進の桁を含みます。

    A decimal value may also include an exponent following the significand, indicating the power of 10 by which the significand should be multiplied. If included, the exponent is separated by a single character, e or E, and consists of an optional plus or minus sign character and a sequence of decimal digits.
    10進の値はまた、それによって仮数が乗算されるべき10の冪を指し示す指数を、仮数に続けて含んでも構いません。もし含むならば、指数は１つの文字eもしくはEによって区切られます、そして随意のプラスまたはマイナス符号文字と一連の10進の桁から成り立ちます。

  • A hexadecimal value contains the significand, either 0X or 0x, followed by a sequence of hexadecimal digits. The significand may include a decimal point.
    16進数値は、仮数、0Xまたは0xのどちらかに続く16進の桁を含みます。仮数は、小数点を含んでも構いません。

    A hexadecimal value may also include an exponent following the significand, indicating the power of 2 by which the significand should be multiplied. If included, the exponent is separated by a single character, p or P, and consists of an optional plus or minus sign character and a sequence of decimal digits.
    16進数値はまた、それによって仮数が乗算されるべき２の冪を指し示す指数を、仮数に続けて含んでも構いません。もし含むならば、指数は１つの文字pもしくはPによって区切られます、そして随意のプラスまたはマイナス符号文字と一連の10進の桁から成り立ちます。

  • A value of infinity contains one of the strings "inf" or "infinity", case insensitive.
    無限大の値は、文字列"inf"または"infinity"のうちの１つを含みます、大文字小文字に影響されずに。

  • A value of NaN contains the string "nan", case insensitive.
    NaNの値は、文字列"nan"を含みます、大文字小文字に影響されずに。

    An NaN value may also include a payload in parentheses following the "nan" keyword. The payload consists of a sequence of decimal digits, or the characters 0X or 0x followed by a sequence of hexadecimal digits. If the payload contains any other characters, it is ignored. If the value of the payload is larger than can be stored as the payload of a Float80.nan, the least significant bits are used.
    NaN値はまた、"nan"キーワードに続けて丸括弧の中にペイロードを含むことができます。ペイロードは、一連の10進の桁、または文字0Xか0xに続く一連の16進の桁から成ります。ペイロードが何か他の文字を含むならば、それは無視されます。ペイロードの値がFloat80.nanのペイロードとして格納可能であるより大きいならば、最下位ビットが使われます。
