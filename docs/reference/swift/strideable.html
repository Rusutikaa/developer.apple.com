Protocol

Strideable

Conforming types are notionally continuous, one-dimensional values that can be offset and measured.
準拠する型は、概念上連続的な、一次元の値で、それはオフセットできて計測できるものです。

Associated Types

Stride

A type that represents the distance between two values of Self.
Selfの２つの値の間の隔たりを表すある型。

Instance Methods
インスタンスメソッド

func advanced(by: Self.Stride)

Returns a Self x such that self.distance(to: x) approximates n.
Self xを返します、self.distance(to: x)はnの近似となります。

func distance(to: Self)

Returns a stride x such that self.advanced(by: x) approximates other.
進み幅xを返します、self.advanced(by: x)はotherの近似となります（otherはラベルtoで指定されるもの？）。
