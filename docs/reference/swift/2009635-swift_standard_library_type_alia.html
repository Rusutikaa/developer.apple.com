Swift Standard Library Type Aliases
Swift標準ライブラリ型エイリアス

This chapter describes the global type aliases defined in the Swift standard library.
この章は、スウィフト標準ライブラリで定義されるグローバルな型エイリアスを記述します。

AnyClass

The protocol to which all class types implicitly conform.
全てのクラスが暗黙的に準拠するプロトコル。

BooleanLiteralType

CChar
