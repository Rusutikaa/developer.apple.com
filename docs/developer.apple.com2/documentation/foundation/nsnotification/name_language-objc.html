NSNotification.Name

The type used for the name of a notification. ある通知の名前のために使われる型。

struct Name

Overview 概要

In Objective-C, NSNotification names are type aliased to the NSString class. In Swift, Notification names use the nested NSNotification.Name type. Objective-Cでは、NSNotification名は、NSStringクラスにエイリアスされる型です。Swiftでは、Notification名は、入れ子にされたNSNotification.Name型を使います。
