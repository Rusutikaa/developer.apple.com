Operator 演算子

>=(_:_:)

Returns a Boolean value indicating whether the value of the first argument is greater than or equal to that of the second argument. 最初の引数の値が２番目の引数のそれより大きいまたは等しいかどうかを指し示すブール値を返します。

SDK

  • Xcode 9.0+

Framework フレームワーク

  • Swift Standard Library Swift標準ライブラリ

On This Page この項には

  • Declaration 宣言
  • Parameters パラメータ
  • Return Value 戻り値
  • Discussion 解説

Declaration 宣言

Parameters パラメータ

lhs

A value to compare. 比較する値。

rhs

Another value to compare. もう一方の比較する値。

Return Value 戻り値

true if lhs is greater than or equal to rhs; otherwise, false. trueを、lhsがrhsより大きいか等しいならば返します；そうでなければ、false。

Discussion 解説

This is the default implementation of the greater-than-or-equal-to operator (>=) for any type that conforms to Comparable. Comparableに準拠するあらゆる型に対して、より大きいか等しい演算子（>=）の初期状態の実装です。
