UInt8

An 8-bit unsigned integer value type. 8ビット符号なし整数値型。

  • macOS 10.10+
  • Mac Catalyst 13.0+
  • tvOS 9.0+
  • watchOS 2.0+
  • Xcode 6.0.1+
