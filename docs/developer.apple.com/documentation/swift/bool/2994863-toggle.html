Instance Method インスタンスメソッド

toggle()

Toggles the Boolean variable’s value. ブール変数の持つ値を切り替えます。

SDK

  • Xcode 10.0+

Framework フレームワーク

  • Swift Standard Library Swift標準ライブラリ

On This Page この項には

  • Declaration 宣言
  • Discussion 解説
  • See Also 参照

Declaration 宣言

Discussion 解説

Use this method to toggle a Boolean value from true to false or from false to true. このメソッドを使って、ブール値をtrueからfalseへとまたはfalseからtrueへと切り替えてください。

var bools = [true, false]

bools[0].toggle() // bools == [false, false]
