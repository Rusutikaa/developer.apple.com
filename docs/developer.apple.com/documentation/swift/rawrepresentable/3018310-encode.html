Instance Method インスタンスメソッド

encode(to:)

Encodes this value into the given encoder, when the type’s RawValue is UInt16. この値をその与えられたデコーダへとデコードします、RawValueがUInt16である場合。

SDK

  • Xcode 10.0+

Framework フレームワーク

  • Swift Standard Library Swift標準ライブラリ

On This Page この項には

  • Declaration 宣言
  • Parameters パラメータ
  • Discussion 解説
  • See Also 参照

Declaration 宣言

Available when Self conforms to Encodable and RawValue is UInt16. SelfがEncodableに準拠してRawValueがUInt16である時に利用可能です。

Parameters パラメータ

encoder

The encoder to write data to. データを書き込むエンコーダ。

Discussion 解説

This function throws an error if any values are invalid for the given encoder’s format. この関数は、与えられたエンコーダの形式に対して何らかの値が無効であるならばエラーをスローします。
