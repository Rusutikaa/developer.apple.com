C Interoperability C互換性

Use imported C types or call C variadic functions. インポートされたC型を使ったり、C可変長引数関数を呼び出します。

Framework フレームワーク

  • Swift Standard Library Swift標準ライブラリ

Topics 話題

C and Objective-C Pointers CおよびObjective-Cポインタ

struct OpaquePointer

A wrapper around an opaque C pointer. 不透明Cポインターの周りのラッパー。

struct AutoreleasingUnsafeMutablePointer

A mutable pointer-to-ObjC-pointer argument. 可変のpointer-to-ObjC-pointer引数。

C Variadic Functions C可変長引数関数

func withVaList<R>([CVarArg], (CVaListPointer) -> R) -> R

Invokes the given closure with a C va_list argument derived from the given array of arguments. 与えられたクロージャを、その与えられた引数の配列から引き出されたC va_list引数で発動します。

struct CVaListPointer

A wrapper around a C va_list pointer. C va_listポインタの周りのラッパー。

protocol CVarArg

A type whose instances can be encoded, and appropriately passed, as elements of a C va_list. それのインスタンスが、符号化されること、そして適切に渡されることが、C va_listの要素として可能である型。

func getVaList([CVarArg]) -> CVaListPointer

Returns a CVaListPointer that is backed by autoreleased storage, built from the given array of arguments. オートリリースされるストレージによって裏付けられるCVaListPointerを返します、いくらかの引数からなる与えられた配列から組み立てられます。

Pointers to Values 値へのポインタ

Access a pointer to a variable in memory for explicit passing to C functions. メモリ中のある変数へのポインタに明示的にC関数を渡すためにアクセスします。

func withUnsafePointer<T, Result>(to: inout T, (UnsafePointer<T>) -> Result) -> Result

Invokes the given closure with a pointer to the given argument. 与えられたクロージャを、与えられた引数に対するポインタとともに発動します。

func withUnsafeMutablePointer<T, Result>(to: inout T, (UnsafeMutablePointer<T>) -> Result) -> Result

Calls the given closure with a mutable pointer to the given argument. 与えられたクロージャを与えられた引数に対する可変ポインタを使って呼び出します。

func withUnsafeBytes<T, Result>(of: inout T, (UnsafeRawBufferPointer) -> Result) -> Result

Invokes the given closure with a buffer pointer covering the raw bytes of the given argument. 与えられたクロージャを、与えられた引数の生のバイトを変換しているバッファポインタとともに発動します。

func withUnsafeMutableBytes<T, Result>(of: inout T, (UnsafeMutableRawBufferPointer) -> Result) -> Result

Invokes the given closure with a mutable buffer pointer covering the raw bytes of the given argument. 与えられたクロージャを、与えられた引数の生のバイトを変換している可変バッファポインタとともに発動します。

Programming Tasks プログラミング作業

Input and Output 入力と出力

Print values to the console, read from and write to text streams, and use command line arguments. 値をコンソールに出力し、テキストストリームから読み込みおよびそれへと書き出し、そしてコマンドライン引数を使います。

Debugging and Reflection デバッグとリフレクション

Fortify your code with runtime checks, and examine your values' runtime representation. あなたのコードを実行時チェックで強化し、そしてあなたの値の実行時表現を検査します。

Key-Path Expressions キーパス式

Use key-path expressions to access properties dynamically. キーパス式を使ってプロパティに動的にアクセスします。

Manual Memory Management 手動メモリ管理

Allocate and manage memory manually. メモリを手動でアロケートして管理します。

Type Casting and Existential Types 型キャストと存在型

Perform casts between types or represent values of any type. 型の間でキャストを実行したり、任意型の値を表します。

Operator Declarations 演算子宣言

Work with prefix, postfix, and infix operators. 前置、後置、および中置演算子を扱います。
