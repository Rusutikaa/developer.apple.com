Global Numeric Functions グローバルな数値関数

Use these functions with numeric values and other comparable types. これらの関数を数値および他の比較可能な型で使ってください。

  • Swift Standard Library Swift標準ライブラリ

Topics 話題

Choosing the Smallest and Largest Value 最小および最大値を選ぶ

func min<T>(T, T) -> T

Returns the lesser of two comparable values. ２つの比較可能な値のうちより小さいものを返します。

func min<T>(T, T, T, T...) -> T

Returns the least argument passed. 渡された引数の最も少ないものを返します。

func max<T>(T, T) -> T

Returns the greater of two comparable values. ２つの比較可能な値のうちより大きいものを返します。

func max<T>(T, T, T, T...) -> T

Returns the greatest argument passed. 渡された引数の最も大きいものを返します。

Finding the Absolute Value 絶対値を見つける

func abs<T>(T) -> T

Returns the absolute value of the given number. 与えられた数の絶対値を返します。
