Input and Output 入力と出力

Print values to the console, read from and write to text streams, and use command line arguments. 値をコンソールに出力し、テキストストリームから読み込みおよびそれへと書き出し、そしてコマンドライン引数を使います。

On This Page この項には

  • Topics 話題
  • See Also 参照

Topics 話題

Text Output テキスト出力

func print(Any, separator: String, terminator: String)

Writes the textual representations of the given items into the standard output. 与えられた項目のテキスト表現を標準出力に書き出します。

func print<Target>(Any, separator: String, terminator: String, to: inout Target)

Writes the textual representations of the given items into the given output stream. 与えられた要素のテキスト表現を与えられた出力ストリームに書き出します。

Command Line Input コマンドライン入力

enum CommandLine

Command-line arguments for the current process. 現在のプロセスに対するコマンドライン引数。

func readLine(strippingNewline: Bool) -> String?

Returns a string read from standard input through the end of the current line or until EOF is reached. 標準入力から現在の行の終わりまでまたはEOFが到達されるまで読み込んだ文字列を返します。

Streams ストリーム

protocol TextOutputStream

A type that can be the target of text-streaming operations. テキストストリーミング操作のターゲットであることができる型。

protocol TextOutputStreamable

A source of text-streaming operations. テキストストリーミング操作のソース。

Programming Tasks プログラミング作業

Debugging and Reflection デバッグとリフレクション

Fortify your code with runtime checks, and examine your values' runtime representation. あなたのコードを実行時チェックで強化し、そしてあなたの値の実行時表現を検査します。

Key-Path Expressions キーパス式

Use key-path expressions to access properties dynamically. キーパス式を使ってプロパティに動的にアクセスします。

Manual Memory Management 手動メモリ管理

Allocate and manage memory manually. メモリを手動でアロケートして管理します。

Type Casting and Existential Types 型キャストと存在型

Perform casts between types or represent values of any type. 型の間でキャストを実行したり、任意型の値を表します。

C Interoperability C互換性

Use imported C types or call C variadic functions. インポートされたC型を使ったり、C可変長引数関数を呼び出します。

Operator Declarations 演算子宣言

Work with prefix, postfix, and infix operators. 前置、後置、および中置演算子を扱います。
