Type Casting and Existential Types 型キャストと存在型

Perform casts between types or represent values of any type. 型の間でキャストを実行したり、任意型の値を表します。

Framework フレームワーク

  • Swift Standard Library Swift標準ライブラリ

On This Page この項には

  • Topics 話題
  • See Also 参照

Topics 話題

Integer Value Casting 整数値のキャスト

func numericCast<T, U>(T) -> U

Returns the given integer as the equivalent value in a different integer type. 与えられた整数を同等値として異なる整数型において返します。

Closure Casting クロージャのキャスト

func withoutActuallyEscaping<ClosureType, ResultType>(ClosureType, do: (ClosureType) -> ResultType) -> ResultType

Allows a nonescaping closure to temporarily be used as if it were allowed to escape. 非脱出クロージャに、一時的にそれが脱出することを許可されたかのように使われることを許可します。

Instance Casting インスタンスのキャスト

func unsafeDowncast<T>(AnyObject, to: T.Type) -> T

Returns the given instance cast unconditionally to the specified type. 与えられたインスタンスを無条件に特定の型にキャストして返します。

func unsafeBitCast<T, U>(T, to: U.Type) -> U

Returns the bits of the given instance, interpreted as having the specified type. 与えられたインスタンスのビットを返します、指定された型を持つと解釈されます。

Existential Types 存在型

Use a variable or constant with an existential type to hold an instance of any type. ある変数または定数を存在型とともに使うことで、どんな型のインスタンスでも保持するようになります。

typealias AnyObject

The protocol to which all classes implicitly conform. このプロトコルに対して、全てのクラスは暗黙的に準拠します。

typealias AnyClass

The protocol to which all class types implicitly conform. このプロトコルに対して、全てのクラス型は暗黙的に準拠します。

Comparing Identity 同一性の比較

func === (AnyObject?, AnyObject?) -> Bool

Returns a Boolean value indicating whether two references point to the same object instance. ２つの参照が同じオブジェクトインスタンスを指しているかどうかを示すブール値を返します。

func !== (AnyObject?, AnyObject?) -> Bool

Returns a Boolean value indicating whether two references point to different object instances. ２つの参照が異なるオブジェクトインスタンスを指しているかどうかを示すブール値を返します。

Void Type void型

typealias Void

The return type of functions that don’t explicitly specify a return type, that is, an empty tuple (). 戻り型を明示的に指定されない関数の戻り型、すなわち、空のタプル()。

Programming Tasks プログラミング作業

Input and Output 入力と出力

Print values to the console, read from and write to text streams, and use command line arguments. 値をコンソールに出力し、テキストストリームから読み込みおよびそれへと書き出し、そしてコマンドライン引数を使います。

Debugging and Reflection デバッグとリフレクション

Fortify your code with runtime checks, and examine your values' runtime representation. あなたのコードを実行時チェックで強化し、そしてあなたの値の実行時表現を検査します。

Key-Path Expressions キーパス式

Use key-path expressions to access properties dynamically. キーパス式を使ってプロパティに動的にアクセスします。

Manual Memory Management 手動メモリ管理

Allocate and manage memory manually. メモリを手動でアロケートして管理します。

C Interoperability C互換性

Use imported C types or call C variadic functions. インポートされたC型を使ったり、C可変長引数関数を呼び出します。

Operator Declarations 演算子宣言

Work with prefix, postfix, and infix operators. 前置、後置、および中置演算子を扱います。
