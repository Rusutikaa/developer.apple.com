Type Aliases 型エイリアス

typealias Int32.Words.Indices
A type that represents the indices that are valid for subscripting the collection, in ascending order. 昇順での、コレクションの添え字として有効なインデックスを表す型。
typealias Int32.Words.Element
A type representing the sequence’s elements. このシーケンスの持つ要素を表している型。
typealias Int32.Words.Index
A type that represents a position in the collection. コレクションの中のある位置を表す型。
typealias Int32.Words.Iterator
A type that provides the collection’s iteration interface and encapsulates its iteration state. そのコレクションの持つ反復インターフェイスを提供してそれの反復状態をカプセル化するある型。
typealias Int32.Words.SubSequence
A sequence that represents a contiguous subrange of the collection’s elements. コレクションの要素の隣接下位範囲を表すシーケンス。
