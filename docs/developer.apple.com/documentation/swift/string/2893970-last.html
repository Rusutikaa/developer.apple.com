last

The last element of the collection. コレクションの最後の要素。

SDK

  • Xcode 9.0+

Framework フレームワーク

  • Swift Standard Library Swift標準ライブラリ

Discussion 解説

If the collection is empty, the value of this property is nil. コレクションが空であるならば、このプロパティはnilです。
