Protocol

SIMDStorage

A SIMD vector type that may not have any computational operations. あるSIMDベクター型、それは何ら計算関係の演算を持たないかもしれません。

SDK

  • Xcode 10.2+Beta

Framework フレームワーク

  • Swift Standard Library Swift標準ライブラリ

Declaration 宣言

Overview 概要

This protocol only defines a storage layout and provides elementwise accesses. Computational operations are defined on SIMDVector, which refines this protocol, or on the concrete types that conform. このプロトコルは、ストレージレイアウトを定義するそして要素関連アクセスを提供するだけです。計算関係演算は、SIMDVector上で定義されます、それはこのプロトコルを洗練させたものです、または準拠する具象型上で。

The type of scalars in the vector space. このベクタースペースの中のスカラーそれらの型。

Required. 必須。

Initializers イニシャライザ

init()

A vector with zero in all lanes. 全てのレーンにおいてゼロをもつあるベクター。

Required. 必須。

Instance Properties 様々なインスタンスプロパティ

var scalarCount: Int

The number of scalars/elements in the vector. このベクターの中のスカラー／要素の数。

Required. 必須。

Subscripts 添え字

subscript(Int) -> Self.Scalar

Element access to the vector. ベクターに対する要素アクセス。

Required. 必須。
