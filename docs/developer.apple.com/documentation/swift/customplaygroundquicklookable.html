Protocol

CustomPlaygroundQuickLookable

A type that explicitly supplies its own playground Quick Look. 明示的にそれ自身のプレイグラウンド「クイックルック」を提供する型。

SDK

  • Xcode 7.0–9.3Deprecated

Framework フレームワーク

  • Swift Standard Library Swift標準ライブラリ

Overview 概要

The CustomPlaygroundQuickLookable protocol is deprecated, and will be removed from the standard library in a future Swift release.To customize the logging of your type in a playground, conform to the CustomPlaygroundDisplayConvertible protocol. CustomPlaygroundQuickLookableプロトコルは推奨されません、将来のSwiftリリースでの標準ライブラリから取り除かれるでしょう。あなたの型のログ取りをプレイグラウンドにおいてカスタマイズするには、CustomPlaygroundDisplayConvertibleプロトコルに準拠してください。

If you need to provide a customized playground representation in Swift 4.0 or Swift 3.2 or earlier, use a conditional compilation block: あなたがカスタマイズされたプレイグラウンド表現をSwift 4.0またはSwift 3.2以前において提供する必要があるならば、条件コンパイルブロックを使ってください：
