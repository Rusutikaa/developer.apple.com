Instance Method インスタンスメソッド

subtract(_:)

Removes the elements of the given set from this set. 与えられた集合の要素をこの集合から削除します。

SDK

  • Xcode 8.0+

Framework フレームワーク

  • Swift Standard Library Swift標準ライブラリ

On This Page この項には

  • Declaration 宣言
  • Parameters パラメータ
  • Discussion 解説
  • See Also 参照

Declaration 宣言

Parameters パラメータ

other

A set of the same type as the current set. 現在の集合と同じ型の集合。

Discussion 解説

In the following example, the elements of the employees set that are also members of the neighbors set are removed. In particular, the names "Bethany" and "Eric" are removed from employees. 次の例において、employees集合の要素でまたneighbors集合のメンバでもあるものは削除されます。個別的には、"Bethany"と"Eric"がemployeesから削除されます。
