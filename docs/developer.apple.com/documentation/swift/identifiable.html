Protocol

Identifiable

A class of types whose instances hold the value of an entity with stable identity. それのインスタンスがある存在の値を安定同一性で保持するところの型それらからなるあるクラス。

SDK

  • Xcode 11.0+

Framework フレームワーク

  • Swift Standard Library Swift標準ライブラリ

Declaration 宣言

Topics 話題

Specifying the Associated Type 関連型を指定する

associatedtype ID

A type representing the stable identity of the entity associated with self. selfと結びつけられた存在の安定同一性を表しているある型。

Required. 必須。

Specifying the Identified Item 同一項目を指定する

var id: Self.ID

The stable identity of the entity associated with self. selfと結びつけられた存在の安定同一性。

Required. Default implementation provided. 必須。 省略時の実装の提供。
