Protocol

RangeExpression

A type that can be used to slice a collection. コレクションをスライスするために使われる型。

SDK

  • Xcode 9.0+

Framework フレームワーク

  • Swift Standard Library Swift標準ライブラリ

Declaration 宣言

Overview 概要

A type that conforms to RangeExpression can convert itself to a Range<Bound> of indices within a given collection. RangeExpressionに準拠する型は、それ自身を指定コレクション内のインデックスのRange<Bound>に変換できます。

Topics 話題

Associated Types さまざまな関連型

associatedtype Bound

The type for which the expression describes a range. それに対してこの式が範囲を記述する型。

Required. 必須。

Instance Methods インスタンスメソッド

func contains(Self.Bound) -> Bool

Returns a Boolean value indicating whether the given element is contained within the range expression. 与えられた要素がこの範囲式内に含まれるかどうかを指し示すブール値を返します。

Required. 必須。

func relative<C>(to: C) -> Range<Self.Bound>

Returns the range of indices described by this range expression within the given collection. 与えられたコレクション内の、この範囲式によって記述されるインデックスの範囲を返します。

Required. 必須。

Operator Functions 演算子関数

static func ~= (Self, Self.Bound) -> Bool

See Also 参照

Range Expressions 範囲式

struct PartialRangeUpTo

A partial half-open interval up to, but not including, an upper bound. 上側の境界までの、しかしそれを含まない部分的半開間隔。

struct PartialRangeThrough

A partial interval up to, and including, an upper bound. ある上側の境界までの、そしてそれを含んでいる部分的な間隔。

struct PartialRangeFrom

A partial interval extending upward from a lower bound. ある下側の境界から上向きに拡張する部分的な間隔。

enum UnboundedRange_

A range expression that represents the entire range of a collection. あるコレクションの全体の範囲を表す範囲式。
