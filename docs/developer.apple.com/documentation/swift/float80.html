Float80

An extended-precision, floating-point value type. 拡張倍精度、浮動小数点値型。

SDK

  • Xcode 6.1+

Topics 話題

Type Aliases 型エイリアス

typealias Float80.Exponent

A type that can represent any written exponent. 何らかの書かれた指数を表すことができる型。

typealias Float80.Magnitude

A type that can represent the absolute value of any possible value of this type. この型の何らかの可能な値の絶対値を表すことのできる型。

typealias Float80.RawSignificand

A type that represents the encoded significand of a value. ある値のエンコードされた仮数を表す型。
