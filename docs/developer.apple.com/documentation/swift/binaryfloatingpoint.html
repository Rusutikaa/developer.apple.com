BinaryFloatingPoint

A radix-2 (binary) floating-point type. あるradix-2（バイナリ）浮動小数点型。

Declaration 宣言

Overview 概要

The BinaryFloatingPoint protocol extends the FloatingPoint protocol with operations specific to floating-point binary types, as defined by the IEEE 754 specification. BinaryFloatingPoint is implemented in the standard library by Float, Double, and Float80 where available. BinaryFloatingPointプロトコルは、FloatingPointプロトコルを、IEEE 754仕様で定義されるとおりに、浮動小数点バイナリ型に特有の演算で拡張します。BinaryFloatingPointは、標準ライブラリにおいてFloat、Double、そして利用可能なところではFloat80によって実装されます。
