NSCalendarDate

A specialized date object with embedded calendar information. 特殊化された日付オブジェクトで、埋め込みカレンダー情報を持ちます。

Deprecated 非推奨

Use NSDate and NSCalendar instead. NSDateとNSCalendarを代わりに使ってください。
