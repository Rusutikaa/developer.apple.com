Declaration 宣言

Overview 概要

You typically use instances of NSUnitMass to represent specific quantities of mass using the NSMeasurement class. あなたは概してNSUnitMassのインスタンスを使用して、NSMeasurementクラスを使っている質量の具体的な量を表します。

Mass 質量

Mass is a fundamental property of matter that causes it to resist being accelerated by a force. The SI unit for mass is the kilogram (kg), which defined in terms of the mass of the international prototype kilogram. 質量は、物質の基本的な物理特性です、それはそれにある力によって加速されることに抵抗させます。質量に対するSI単位は、キログラム（kg）です、それは国際キログラム原器の質量に基づいて定義されます。

The NSUnitMass class defines its base unit as kilograms, and provides the following units, which are initialized using UnitConverterLinear converters with the specified coefficients: NSUnitMassクラスは、それの基本単位をkilogramsとして定義します、そして以下の単位を提供します、それらは指定された係数でUnitConverterLinear変換器を使用して初期化されます：
