NSKeyValueObserving

An informal protocol that objects adopt to be notified of changes to the specified properties of other objects. ある非形式プロトコル、それは、オブジェクトが採用して、他のオブジェクトの指定されたプロパティに対する変更の通知を受けるようになるものです。

Framework フレームワーク

  • Foundation

On This Page この項には

  • Overview 概要
  • Topics 話題
  • See Also 参照

Overview 概要

You can observe any object properties including simple attributes, to-one relationships, and to-many relationships. Observers of to-many relationships are informed of the type of change made — as well as which objects are involved in the change. あなたは、単純な属性、対一関係、そして対多関係を含めて、あらゆるオブジェクトプロパティを監視できます。対多関係の監視は、なされる変更の種類 — それだけでなくどのオブジェクトが変更に関わっているか、を通知されます。

NSObject provides an implementation of the NSKeyValueObserving protocol that provides an automatic observing capability for all objects. You can further refine notifications by disabling automatic observer notifications and implementing manual notifications using the methods in this protocol. NSObjectは、NSKeyValueObservingプロトコルのある実装を提供します、それはすべてのオブジェクトに対して自動監視能力を提供します。あなたは、さらにすすめて通知を改良することが、自動監視通知を作動しないようにして手動通知をこのプロトコルのメソッドを使って実装することによって可能です。

Topics 話題

Change Notification 通知を変更する

func observeValue(forKeyPath: String?, of: Any?, change: [NSKeyValueChangeKey : Any]?, context: UnsafeMutableRawPointer?)

Informs the observing object when the value at the specified key path relative to the observed object has changed. 監視されるオブジェクトに相対的な、指定されるキーパスでの値が変化した時に、監視されるオブジェクトに知らせます。

Registering for Observation 監視のために登録する

func addObserver(NSObject, forKeyPath: String, options: NSKeyValueObservingOptions, context: UnsafeMutableRawPointer?)

Registers the observer object to receive KVO notifications for the key path relative to the object receiving this message. オブザーバーオブジェクトを登録して、このメッセージを受け取っているオブジェクトに相対的なキーパスに関するKVO通知を受け取るようにします。

func removeObserver(NSObject, forKeyPath: String)

Stops the observer object from receiving change notifications for the property specified by the key path relative to the object receiving this message. オブザーバーオブジェクトが、このメッセージを受け取っているオブジェクトに相対的なキーパスによって指定されるプロパティに関する変更通知を受け取ることを止めます。

func removeObserver(NSObject, forKeyPath: String, context: UnsafeMutableRawPointer?)

Stops the observer object from receiving change notifications for the property specified by the key path relative to the object receiving this message, given the context. オブザーバーオブジェクトが、与えられた文脈で、このメッセージを受け取っているオブジェクトに相対的なキーパスによって指定されるプロパティに関する変更通知を受け取ることを止めます。

Notifying Observers of Changes 変更のオブザーバに通知する

func willChangeValue(forKey: String)

Informs the observed object that the value of a given property is about to change. 与えられたプロパティの値が変更されようとしていることを監視されるオブジェクトに知らせます。

func didChangeValue(forKey: String)

Informs the observed object that the value of a given property has changed. 与えられたプロパティの値が変更されたことを監視されるオブジェクトに知らせます。

func willChange(NSKeyValueChange, valuesAt: IndexSet, forKey: String)

Informs the observed object that the specified change is about to be executed at given indexes for a specified ordered to-many relationship. 指定された変更がいくつかの与えられたインデックスである指定された順序付けられた対多関係に対して実行されようとしていることを監視されるオブジェクトに知らせます。

func didChange(NSKeyValueChange, valuesAt: IndexSet, forKey: String)

Informs the observed object that the specified change has occurred on the indexes for a specified ordered to-many relationship. 指定された変更がいくつかの与えられたインデックスである指定された順序付けられた対多関係に対して起こったことを監視されるオブジェクトに知らせます。

func willChangeValue(forKey: String, withSetMutation: NSKeyValueSetMutationKind, using: Set<AnyHashable>)

Informs the observed object that the specified change is about to be made to a specified unordered to-many relationship. 指定された変更がある指定された順序付けられない対多関係に対してされようとしていることを監視されるオブジェクトに知らせます。

func didChangeValue(forKey: String, withSetMutation: NSKeyValueSetMutationKind, using: Set<AnyHashable>)

Informs the observed object that the specified change was made to a specified unordered to-many relationship. 指定された変更がある指定された順序付けられない対多関係に対してされたことを監視されるオブジェクトに知らせます。

Observing Customization カスタマイズを監視する

class func automaticallyNotifiesObservers(forKey: String) -> Bool

Returns a Boolean value that indicates whether the observed object supports automatic key-value observation for the given key. 監視されるオブジェクトが自動キー値監視を与えられたキーに対してサポートするかどうかを指し示すブール値を返します。

class func keyPathsForValuesAffectingValue(forKey: String) -> Set<String>

Returns a set of key paths for properties whose values affect the value of the specified key. それらの値が指定されたキーの値に影響を与えるプロパティいくらかに対するキーパスひとそろいを返します。

protocol NSKeyValueObservingCustomization
var observationInfo: UnsafeMutableRawPointer?

Returns a pointer that identifies information about all of the observers that are registered with the observed object. 監視対象オブジェクトに登録される全てのオブザーバーについての情報を識別するポインタを返します。

Constants 定数

class NSKeyValueObservation
struct NSKeyValueObservedChange
enum NSKeyValueChange

The kinds of changes that can be observed. 監視されることが可能である、変更の種類。

struct NSKeyValueObservingOptions

The values that can be returned in a change dictionary. ある変更辞書において返されることが可能な値。

struct NSKeyValueChangeKey

The keys that can appear in the change dictionary. 変更辞書において現れることができるキー。

enum NSKeyValueSetMutationKind

The kinds of mutation that you can make to an unordered collection. 変化のさまざまな種類、それはあなたが順不同コレクションにすることができます。
