current

Declaration 宣言

Return Value 戻り値

The logical calendar for the current user. 現在のユーザに対する論理カレンダー。

Discussion 議論

The returned calendar is formed from the settings for the current user’s chosen system locale overlaid with any custom settings the user has specified in System Preferences. Settings you get from this calendar do not change as System Preferences are changed, so that your operations are consistent (contrast with autoupdatingCurrent). 返されるカレンダーは、ユーザがシステム環境設定で指定した任意のあつらえの設定でオーバーレイされる、現在のユーザの選択システムロケールから形成されます。あなたがこのカレンダーから得る設定は、システム環境設定が変更されると同時に変わりません、なのであなたの演算は首尾一貫したものになります（autoupdatingCurrentとは対照的に）。
