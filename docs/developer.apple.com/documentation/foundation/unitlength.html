Declaration 宣言

Overview 概要

You typically use instances of NSUnitLength to represent specific quantities of length using the NSMeasurement class. あなたは概してNSUnitLengthのインスタンスを使用して、NSMeasurementクラスを使っている長さの具体的な量を表します。

Length 長さ

Length is the dimensional extent of matter. The SI unit for length is the meter (m), which is defined in terms of the distance traveled by light in a vacuum. 長さは、物質の寸法的な広がりです。長さに対するSI単位はメートル（m）です、それは真空中の光が進む距離に基づいて定義されます。

The NSUnitLength class defines its base unit as meters, and provides the following units, which are initialized using UnitConverterLinear converters with the specified coefficients: NSUnitLengthクラスは、それの基本単位をmetersとして定義します、そして以下の単位を提供します、それらは指定された係数でUnitConverterLinear変換器を使用して初期化されます：
