Instance Method インスタンスメソッド

isSuperset(of:)

Returns a Boolean value that indicates whether the set is a superset of the given set. この集合が、与えられた集合の上位集合であるかどうかを指し示すブール値を返します。

SDKs

  • iOS 7.0+
  • macOS 10.9+
  • Mac Catalyst 13.0+
  • tvOS 9.0+
  • watchOS 2.0+
  • Xcode 8.0+

Framework フレームワーク

  • Foundation

Declaration 宣言

Parameters パラメータ

other

A set of the same type as the current set. 現在の集合と同じ型の集合。

Return Value 戻り値

true if the set is a superset of other; otherwise, false. この集合がotherの上位集合であるならばtrue；そうでなければ、false。

Discussion 議論

Set A is a superset of another set B if every member of B is also a member of A. 集合Aがもう１つの集合Bの狭義の上位集合であるのは、BのすべてのメンバがまたAのメンバである場合です。
