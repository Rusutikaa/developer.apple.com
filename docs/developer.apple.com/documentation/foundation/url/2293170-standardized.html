Instance Property インスタンスプロパティ

standardized

A version of the URL with any instances of “..” or “.” removed from its path. URLのある変形版、それのパスから “..” または “.” のあらゆる出現を取り除いた状態で。

SDKs

  • iOS 7.0+
  • macOS 10.9+
  • Mac Catalyst 13.0+
  • tvOS 9.0+
  • watchOS 2.0+
  • Xcode 8.0+

Accessing the Parts of a URL URLの部分にアクセスする

var absoluteString: String

The absolute string for the URL. そのURLに対する完全な文字列。

var absoluteURL: URL

The absolute URL. 完全URL。

var baseURL: URL?

The base URL. 基準URL。

var fragment: String?

The fragment component of the URL if the URL conforms to RFC 1808 (the most common form of URL), otherwise nil. URLのフラグメント構成要素、もしそのURLがRFC 1808（URLの最も一般的な形式）に準拠するならば、そうでなければnil。

var host: String?

The host component of a URL if the URL conforms to RFC 1808 (the most common form of URL), otherwise nil. URLのホスト構成要素、もしそのURLがRFC 1808（URLの最も一般的な形式）に準拠するならば、そうでなければnil。

var lastPathComponent: String

The last path component of the URL, or an empty string if the path is an empty string. URLの最後のパス構成要素、またはパスが空文字列ならば空文字列。

var path: String

The path component of the URL if the URL conforms to RFC 1808 (the most common form of URL), otherwise an empty string. URLのパス構成要素、もしそのURLがRFC 1808（URLの最も一般的な形式）に準拠するならば、そうでなければ空文字列。

var pathComponents: [String]

The path components of the URL, or an empty array if the path is an empty string. URLのパス構成要素それら、またはパスが空文字列ならば空文字列。

var pathExtension: String

The path extension of the URL, or an empty string if the path is an empty string. URLのパス拡張子構成要素、またはパスが空文字列ならば空文字列。

var port: Int?

The port component of the URL if the URL conforms to RFC 1808 (the most common form of URL), otherwise nil. URLのポート構成要素、もしそのURLがRFC 1808（URLの最も一般的な形式）に準拠するならば、そうでなければnil。

var query: String?

The query of the URL if the URL conforms to RFC 1808 (the most common form of URL), otherwise nil. URLのクエリ、もしそのURLがRFC 1808（URLの最も一般的な形式）に準拠するならば、そうでなければnil。

var relativePath: String

The relative path of the URL if the URL conforms to RFC 1808 (the most common form of URL), otherwise nil. URLの相対パス、もしそのURLがRFC 1808（URLの最も一般的な形式）に準拠するならば、そうでなければnil。

var relativeString: String

The relative portion of a URL. あるURLの相対部分。

var scheme: String?

The scheme of the URL. URLのスキーム。

var standardizedFileURL: URL

A standardized version of the path of a file URL. ファイルURLのパスの標準化バージョン。

var user: String?

The user component of the URL if the URL conforms to RFC 1808 (the most common form of URL), otherwise nil. URLのユーザ構成要素、もしそのURLがRFC 1808（URLの最も一般的な形式）に準拠するならば、そうでなければnil。

var password: String?

The password component of the URL if the URL conforms to RFC 1808 (the most common form of URL), otherwise nil. URLのパスワード構成要素、もしそのURLがRFC 1808（URLの最も一般的な形式）に準拠するならば、そうでなければnil。
