Converting Between Swift Data Types and Strings Swiftデータ型と文字列の間で変換する

Use formatters to convert Swift data types to their string representations and back. フォーマッタを使ってSwiftデータ型をそれらの文字列表現に変換したり戻したりします。

Download

SDK

  • Xcode 10.2+

Framework フレームワーク

  • Foundation

Overview 概要

This sample illustrates how to use the various formatters available in Swift. Using formatters makes it easy to convert Swift data types into readable strings. You can also use formatters to turn strings representing numbers, dates, and measurements into Swift objects that can be manipulated. The strings are locale-aware, which is important because it allows you to localize your content based on region to support different languages, writing formats, and conventions. The playground demonstrates the conversion between Swift data types and their respective string representations based on different formatter styles and locales. このサンプルは、Swiftで利用可能なさまざまなフォーマッタを使う方法を図解します。フォーマッタの使用は、Swiftデータ型を読みやすい文字列へと変換することを簡単にします。あなたはまたフォーマッタを使用して、数、日付、そして寸法を表している文字列を、処理されることが可能なSwiftオブジェクトに変えることができます。文字列はそれらはロケールを認識します、それは重要です、なぜならそれはあなたに異なる言語、書き出し書式設定、そして慣習をサポートするためにあなたのコンテンツを地域に基づきローカライズできるようにするからです。プレイグラウンドは、Swiftデータ型とそれらの各自の文字列表現との間の変換を異なるフォーマッタスタイルとロケールに基づいて実演します。

This playground covers: このプレイグラウンドは以下を扱います：

  • NumberFormatter: Provides localized representations of units and measurements. NumberFormatter：単位と寸法のローカライズされた表現を提供します。

  • PersonNameComponentsFormatter: Provides localized representations of the components of a person’s name. PersonNameComponentsFormatter：ある個人の名前の構成要素それらのローカライズされた表現を提供します。

  • DateFormatter: Converts between dates and their textual representations. DateFormatter：日付とそれらのテキスト表現の間で変換します。

  • DateComponentsFormatter: Creates string representations of quantities of time. DateComponentsFormatter：時間の量の文字列表現を作成します。

  • DateIntervalFormatter: Creates string representations of time intervals. DateIntervalFormatter：時間間隔の文字列表現を作成します。

  • ISO8601DateFormatter: Converts between dates and their ISO 8601 string representations. ISO8601DateFormatter：日付とそれらのISO 8601文字列表現の間で変換します。

  • ByteCountFormatter: Converts a byte count value into a localized description that is formatted with the appropriate byte modifier (KB, MB, GB, etc.). ByteCountFormatter：バイト計数値をローカライズされた記述に変換します、それは適切なバイト修飾子で書式設定されます（KB、MB、GB、など）。

  • MeasurementFormatter: Provides localized representations of units and measurements. MeasurementFormatter：単位と寸法のローカライズされた表現を提供します。

For detailed information about all the available formatters as well as their properties and methods, see Data Formatting. 全ての利用可能なフォーマッタに加えてそれらのプロパティとメソッドについての詳細な情報として、データ書式設定を見てください。
