Calendar.Identifier.islamicTabular

Identifier for the tabular Islamic calendar.

SDKs

  • iOS 8.0+
  • macOS 10.10+
  • Mac Catalyst 13.0+
  • tvOS 9.0+
  • watchOS 2.0+
  • Xcode 6.0.1+

Declaration 宣言

Discussion 議論

This calendar uses arithmetical calculations of months, and is based on the astronomical epoch (Thursday, July 15, 622 CE).
