Protocol

ReferenceConvertible

A decoration applied to types that are backed by a Foundation reference type. Foundation参照型によって裏付けされる型に適用される装飾。

Declaration 宣言

Overview 概要

All ReferenceConvertible types are hashable, equatable, and provide description functions. 全てのReferenceConvertible型はハッシュ化可能、同等性比較可能です、そして記述関数を提供します。
