leftMargin

Keys キー

static let author: NSAttributedString.DocumentAttributeKey

The value of this attribute is an NSString object containing author name. この属性の値は、著者名を含んでいるNSStringオブジェクトです。

static let backgroundColor: NSAttributedString.DocumentAttributeKey

The value of this attribute is an NSColor object representing the document-wide page background color. この属性の値は、NSColorオブジェクトです、書類全体のページ背景色を表しています。

static let bottomMargin: NSAttributedString.DocumentAttributeKey

The value of this attribute is an NSNumber object containing a float, in points. この属性の値は、ポイント数で、あるfloatを含んでいるNSNumberオブジェクトです。

static let category: NSAttributedString.DocumentAttributeKey

The value of this attribute is an NSString object containing the document’s category. この属性の値は、その書類の持つ分類上の区分を含んでいるNSStringオブジェクトです。

static let characterEncoding: NSAttributedString.DocumentAttributeKey

The value of this attribute is an NSNumber object containing integer specifying NSStringEncoding for the file; default for plain text is the default encoding. This key in options can specify the string encoding for reading the data. Upon return, the document attributes can contain the actual encoding used. For writing methods, this value is used for generating the plain text data. この属性の値は、NSNumberオブジェクトで、NSStringEncodingをこのファイルに対して指定している整数を含んでいます；プレーンテキストに対する初期設定は、初期設定のエンコーディングです。このキーはオプションにおいて、データ読み出し用の文字列エンコーディングを指定できます。戻りにおいて、書類属性は実際の使用されたエンコーディングをを含むことができます。書き出しメソッに対して、この値はプレーンテキストデータを生成するために使われます。

static let cocoaVersionDocumentAttribute: NSAttributedString.DocumentAttributeKey

The value of this attribute is an NSNumber object containing a float. For RTF files only, stores the version of Cocoa with which the file was created. Absence of this value indicates RTF file not created by Cocoa or its predecessors. この属性の値は、あるfloatを含んでいるNSNumberオブジェクトです。RTFファイルに対してのみ、それでそのファイルが作成されたCocoaのバージョンを格納します。この値の欠如は、RTFファイルがCocoaまたはそれの前身によって作成されないことを指し示します。

static let comment: NSAttributedString.DocumentAttributeKey

The value of this attribute is an NSString object containing document comments. この属性の値は、書類コメントを含んでいるNSStringオブジェクトです。

static let company: NSAttributedString.DocumentAttributeKey

The value of this attribute is an NSString object containing company or organization name. この属性の値は、企業または組織名を含んでいるNSStringオブジェクトです。

static let converted: NSAttributedString.DocumentAttributeKey

The value of this attribute is an NSNumber object containing an int. Indicates whether the file was converted by a filter service. この属性の値は、intを含んでいるNSNumberオブジェクトです。ファイルがフィルタサービスによって変換されたかどうかを指し示します。

static let copyright: NSAttributedString.DocumentAttributeKey

The value of this attribute is an NSString object containing document copyright info. この属性の値は、書類著作権情報を含んでいるNSStringオブジェクトです。

static let creationTime: NSAttributedString.DocumentAttributeKey

The value of this attribute is an NSDate object containing the creation date of the document; note that this is not the file system creation date of the file, but of the document. この属性の値は、書類の作成日を含んでいるNSDateオブジェクトです；これはファイルのファイルシステム作成日ではなく、書類のであることに注意してください。

static let defaultAttributes: NSAttributedString.DocumentAttributeKey
static let defaultTabInterval: NSAttributedString.DocumentAttributeKey

The value of this attribute is an NSNumber object containing a float. Represents the document-wide default tab stop interval. この属性の値は、あるfloatを含んでいるNSNumberオブジェクトです。書類全体の初期状態のタブ幅を表します。

static let documentType: NSAttributedString.DocumentAttributeKey
static let editor: NSAttributedString.DocumentAttributeKey

The value of this attribute is an NSString object containing name of person who last edited the document. この属性の値は、最後に書類を編集した人物の名を含んでいるNSStringオブジェクトです。

static let excludedElements: NSAttributedString.DocumentAttributeKey

An NSArray object containing NSString objects, representing HTML elements not to be used in generated HTML. あるNSArrayオブジェクト、いくつかのNSStringオブジェクトを含んでいます、それらは生成されたHTMLにおいて使われないHTML要素を表しています。

static let fileType: NSAttributedString.DocumentAttributeKey

The value of this attribute is an NSString object indicating which document type was used to interpret the document, specified as a UTI; for reading, this is available along with documentType, but for writing the two are mutually exclusive. この属性の値は、あるNSStringオブジェクトで、どの書類型がこの書類を解釈するのに使われたかを指し示しています、UTIとして指定されます；読み出しに対して、これはdocumentTypeと一緒に利用可能です、しかし書き出しに対してこの２つは相互に排他的です。

static let hyphenationFactor: NSAttributedString.DocumentAttributeKey

The value of this attribute is an NSNumber object containing a float; 0 = off, 1 = full hyphenation. この属性の値は、あるfloatを含んでいるNSNumberオブジェクトです；0 = オフ、1 = 完全なハイフネーション。

static let keywords: NSAttributedString.DocumentAttributeKey

The value of this attribute is an NSArray object containing NSString objects, containing keywords. この属性の値は、あるNSArrayオブジェクトで、キーワードを含んでいるNSStringオブジェクトいくらかを含んでいます。

static let manager: NSAttributedString.DocumentAttributeKey

The value of this attribute is an NSString object containing the name of the author's manager. この属性の値は、著作者の責任者の名前を含んでいるNSStringオブジェクトです。

static let modificationTime: NSAttributedString.DocumentAttributeKey

The value of this attribute is an NSDate object containing the modification date of the document contents. この属性の値は、その書類の内容の修正日を含んでいるNSDateオブジェクトです。

static let paperMargin: NSAttributedString.DocumentAttributeKey
static let paperSize: NSAttributedString.DocumentAttributeKey

The value of this attribute is an NSValue object containing NSSize. この属性の値は、NSValueオブジェクトで、NSSizeを含んでいます。

static let prefixSpaces: NSAttributedString.DocumentAttributeKey

An NSNumber object containing an integer (default 0) representing the number of spaces per level by which to indent certain nested HTML elements. あるNSNumberオブジェクト、整数（初期状態０）を含んでいます、レベル毎の空白の数を表しています、それによって特定の入れ子にされたHTML要素を字下げすることになります。

static let readOnly: NSAttributedString.DocumentAttributeKey

The value of this attribute is an NSNumber object containing an integer. If missing or 0 or negative, not read only; 1 or more, read only. この属性の値は、integerを含んでいるNSNumberオブジェクトです。見当たらないまたは０または負ならば、読出し専用ではありません；1またはそれ以上は、読出し専用です。

static let rightMargin: NSAttributedString.DocumentAttributeKey

The value of this attribute is an NSNumber object containing a float, in points. この属性の値は、ポイント数で、あるfloatを含んでいるNSNumberオブジェクトです。

static let subject: NSAttributedString.DocumentAttributeKey

The value of this attribute is an NSString object containing subject of document. この属性の値は、書類の主題を含んでいるNSStringオブジェクトです。

static let textEncodingName: NSAttributedString.DocumentAttributeKey

An NSString object containing the name, IANA or otherwise, of a text encoding to be used; mutually exclusive with characterEncoding. あるNSStringオブジェクト、IANAまたはそうでなければ、使われることになるテキストエンコーディングの名前を含んでいます；characterEncodingと相互に排他的です。

static let textLayoutSections: NSAttributedString.DocumentAttributeKey
static let title: NSAttributedString.DocumentAttributeKey

The value of this attribute is an NSString object containing document title. この属性の値は、書類の題名を含んでいるNSStringオブジェクトです。

static let topMargin: NSAttributedString.DocumentAttributeKey

The value of this attribute is an NSNumber object containing a float, in points. この属性の値は、ポイント数で、あるfloatを含んでいるNSNumberオブジェクトです。

static let viewMode: NSAttributedString.DocumentAttributeKey

The value of this attribute is an NSValue object containing an int; 0 = normal; 1 = page layout (use value of @"PaperSize"). この属性の値は、intを含んでいるNSValueオブジェクトです；0 = 通常；1 = ページレイアウト（@"PaperSize"の値を使います）。

static let viewSize: NSAttributedString.DocumentAttributeKey

The value of this attribute is an NSValue object containing NSSize. この属性の値は、NSSizeを含んでいるNSValueオブジェクトです。

static let viewZoom: NSAttributedString.DocumentAttributeKey
