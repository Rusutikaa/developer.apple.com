Type Property 型プロパティ

editor

The name of person who last edited the document. 書類を最後に編集した人物の名前。

Availability 有効性

  • macOS 10.4+

Framework フレームワーク

  • AppKit

On This Page この項には

  • Declaration  宣言 
  • Discussion 
  • See Also  参照 

Declaration 宣言 

static let editor: NSAttributedString.DocumentAttributeKey

Discussion 議論

The value of this attribute is an NSString object containing name of person who last edited the document. この属性の値は、最後に書類を編集した人物の名を含んでいるNSStringオブジェクトです。
