XMLParser.ErrorCode

The following error codes are defined by NSXMLParser. For error codes not listed here, see the <libxml/xmlerror.h> header file. 以下のエラーコードは、NSXMLParserによって定義されます。ここで一覧にされないエラーコードについて、<libxml/xmlerror.h>ヘッダファイルを見てください。
