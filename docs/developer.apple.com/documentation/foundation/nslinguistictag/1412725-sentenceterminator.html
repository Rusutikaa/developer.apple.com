static let otherWord: NSLinguisticTag

This token is a word other than a kind described by other lexical classes (noun, verb, adjective, adverb, pronoun, determiner, particle, preposition, number, conjunction, interjection, classifier, and idiom). このトークンは、他の語彙クラス（名詞、動詞、形容詞、代名詞、限定詞、小詞、前置詞、数詞、接続詞、感嘆詞、分類詞、そして慣用句）によって記述される種類のものとは違う、ある単語です。

static let otherPunctuation: NSLinguisticTag

This token is punctuation other than a kind described by other lexical classes (sentence terminator, open or close quote, open or close parenthesis, word joiner, and dash). このトークンは、他の語彙クラス（文終端子、開きまたは閉じ引用符、開きまたは閉じ括弧、単語結合子、そしてダッシュ記号）によって記述される種類のものとは違う、句読点です。
