Type Property 型プロパティ

eventTracking

A run loop should be set to this mode when tracking events modally, such as a mouse-dragging loop. 実行ループは、イベントをモーダルに追跡している場合はこのモードに設定されるべきです、たとえばマウスドラッグループなど。

Managing the Event Loop イベントループを管理する

func nextEvent(matching: NSEvent.EventTypeMask, until: Date?, inMode: RunLoop.Mode, dequeue: Bool) -> NSEvent?

Returns the next event matching a given mask, or nil if no such event is found before a specified expiration date. ある与えられたマスクに合致している次のイベントを返します、またはnil、もしそのようなイベントがある指定された有効期限の前に見つけられないならば。

func discardEvents(matching: NSEvent.EventTypeMask, before: NSEvent?)

Removes all events matching the given mask and generated before the specified event. 与えられたマスクに合致しているそして指定されたイベントの前に生成された全てのイベントを取り除きます。

var currentEvent: NSEvent?

The last event object that the app retrieved from the event queue. アプリがイベントキューから取って来た最後のイベントオブジェクト。

var isRunning: Bool

A Boolean value indicating whether the main event loop is running. メインイベントループが動作しているかどうかを指し示しているブール値。

func run()

Starts the main event loop. メインイベントループを開始します。

func finishLaunching()

Activates the app, opens any files specified by the NSOpen user default, and unhighlights the app’s icon. アプリを活性化します、NSOpenユーザ初期設定によって指定されたあらゆるファイルを開きます、そしてアプリのもつアイコンの強調表示をやめます。

func stop(Any?)

Stops the main event loop. メインイベントループを停止します。

func sendEvent(NSEvent)

Dispatches an event to other objects. あるイベントを他のオブジェクトへと発送します。

func postEvent(NSEvent, atStart: Bool)

Adds a given event to the receiver’s event queue. 与えられたイベントをレシーバのもつイベントキューに加えます。
