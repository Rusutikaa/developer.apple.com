Measurement

A numeric quantity labeled with a unit of measure, with support for unit conversion and unit-aware calculations. 寸法のいち単位でラベルをつけられる数量、単位変換および単位認識計算に対するサポートを持ちます。

Declaration 宣言

Overview 概要

Measurements support a large set of operators, including +, -, *, /, and a full set of comparison operators. Measurementそれらは、たくさんの演算子セットを、+、-、*、/を含めて、そして比較演算子セットをサポートします。
