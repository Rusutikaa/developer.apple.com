decimalSeparator

The decimal separator associated with the locale. このロケールと結び付けられた小数分離子。

SDKs

  • iOS 2.0+
  • macOS 10.4+
  • Mac Catalyst 13.0+
  • tvOS 9.0+
  • watchOS 2.0+

Declaration 宣言

Discussion 議論

The corresponding value is an NSString object; for example, "." or ",". 対応する値は、NSStringオブジェクトです；例えば、"."、または","。
