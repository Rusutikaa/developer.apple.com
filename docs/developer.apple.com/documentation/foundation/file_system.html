File System ファイルシステム

Create, read, write, and examine files and folders in the file system. ファイルシステムにおいてファイルとフォルダの作成、読み込み、書き出し、そして調査をします。

Topics 話題

File System Operations ファイルシステム演算

About Apple File System Appleファイルシステムについて

Use high-level APIs to get the most out of Apple File System. Appleファイルシステムを最大限に活用するために高水準APIを使ってください。

class FileManager

A convenient interface to the contents of the file system, and the primary means of interacting with it. ファイルシステムの内容、そしてそれと相互作用することの主要な手段に対する便利なインターフェイス。

protocol FileManagerDelegate

The interface a file manager's delegate uses to intervene during operations or if an error occurs. オペレーションの間にまたはエラーが起こる場合に介入するために、ファイルマネージャの委任先が使うインターフェイス。

Managed File Access ファイルアクセスを管理する

class FileHandle

An object-oriented wrapper for a file descriptor. ファイル記述子のためのオブジェクト指向のラッパー。

class NSFileSecurity

A stub class that encapsulates security information about a file. あるファイルについての保安情報をカプセル化するスタブクラス。

class NSFileVersion

A snapshot of a file at a specific point in time. 特定の時点でのあるファイルのスナップショット。

class FileWrapper

A representation of a node (a file, directory, or symbolic link) in the file system. ファイルシステムの中のあるノード（あるファイル、ディレクトリ、またはシンボリックリンク）の表現。

Shared Files 共有ファイル

protocol NSFilePresenter

The interface a file coordinator uses to inform an object presenting a file about changes to that file made elsewhere in the system. あるファイルを表しているオブジェクトに、そのファイルにシステムのどこか他で為される変更について告知するために、ファイルコーディネータが使うインターフェイス。

class NSFileAccessIntent

The details of a coordinated-read or coordinated-write operation. 協調読み込みまたは協調書き出し演算の詳細。

class NSFileCoordinator

An object that coordinates the reading and writing of files and directories among file presenters. ファイルプレゼンタそれらの間でファイルやディレクトリの読み出しや書き込みを調整するオブジェクト。

Errors エラー

File System Error Codes ファイルシステムエラーコード

Recognize common error codes generated by file system operations. ファイルシステム演算によって生成される一般的エラーコードを識別する。

See Also 参照

Files and Data Persistence ファイルとデータ永続

Archives and Serialization アーカイブとシリアライゼーション

Convert objects and values to and from property list, JSON, and other flat binary representations. オブジェクトと値をプロパティリスト、JSON、そして平坦なバイナリ表現へとおよびそれから変換します。

Preferences 環境設定

Persistently store domain-scoped pieces of information used to configure your app. あなたのアプリを設定するために使われる情報のドメインスコープの断片を永続的に格納します。

Spotlight

Search for files and other items on the local device, and index your app's content for searching. ローカルデバイス上のファイルや他の項目を検索します、そしてあなたのアプリのもつ内容を検索のために索引化します。

iCloud

Manage files and key-value data that are automatically synchronized among a user's iCloud devices. ユーザ所有のiCloudデバイス間で自動的に同期するファイルやキー値データを管理します。
