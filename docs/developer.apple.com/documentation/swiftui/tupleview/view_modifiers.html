View Modifiers ビュー修飾子

Configure this view and the views it contains by applying standard modifiers. このビューとそれが含むいくつかのビューを標準修飾子を適用することによって構成設定します。

Framework フレームワーク

  • SwiftUI

On This Page この項には

  • Overview 概要
  • Topics 話題

Overview 概要

For information about how to build and configure views, see the View protocol. どのようにビューをビルドおよび構成設定するかについての情報として、Viewプロトコルを見てください。

Topics 話題

Sizing a View ビューをある大きさに作る

func frame(width: CGFloat?, height: CGFloat?, alignment: Alignment) -> View

Positions the view within an invisible frame with the specified size. ビューを不可視フレーム内でこの指定された大きさで位置決めします。

func frame(minWidth: CGFloat?, idealWidth: CGFloat?, maxWidth: CGFloat?, minHeight: CGFloat?, idealHeight: CGFloat?, maxHeight: CGFloat?, alignment: Alignment) -> View

Positions the view within an invisible frame with the specified width and height. ビューを不可視フレーム内でこの指定された幅と高さで位置決めします。

func fixedSize() -> View

Fixes the view at its ideal size. ビューをそれの最適な大きさに固定します。

func fixedSize(horizontal: Bool, vertical: Bool) -> View

Fixes the view at its ideal size in the specified dimensions. 指定された寸法において、ビューをそれの最適な大きさに固定します。

func layoutPriority(Double) -> View

Sets the priority by which a parent layout should apportion space to the child. それによって親レイアウトが空間を子に配分すべきところの優先度を設定します。

Positioning a View ビューの位置決めをする

func position(CGPoint) -> View

Fixes the center of the view at the specified point in its parent’s coordinate space. ビューの中心をこの指定された点でそれの親のもつ座標空間において固定します。

func position(x: CGFloat, y: CGFloat) -> View

Fixes the center of the view at the specified coordinates in its parent’s coordinate space. ビューの中心をこの指定された座標でそれの親のもつ座標空間において固定します。

func offset(CGSize) -> View

Offsets the view by the horizontal and vertical distances in the given size. ビューをこの与えられた大きさにおける水平および垂直間隔だけ埋め合わせます。

func offset(x: CGFloat, y: CGFloat) -> View

Offsets the view by the specified horizontal and vertical distances. ビューをこの指定された水平および垂直間隔だけ埋め合わせます。

func edgesIgnoringSafeArea(Edge.Set) -> View

Extends the view out of the safe area on the specified edges. ビューをこの指定された縁上の安全領域の外に拡張します。

func coordinateSpace<T>(name: T) -> View

Assigns a name to this view’s coordinate space to which descendants of this view can refer. ある名前をこのビューのもつ座標空間に割り当てます、それに対してこのビューの子孫が参照可能です。

Adjusting Text in a View テキストをビューに合わせる

func keyboardType(UIKeyboardType) -> View

Sets the keyboard type for this view. キーボード型をこのビューに対して設定します。

func font(Font?) -> View

Sets the default font for text in the view. 省略時のフォントをビューの中のテキストに対して設定します。

func lineLimit(Int?) -> View

Sets the maximum number of lines that text can occupy in the view. テキストがビューにおいて占めることができる最大の行数を設定します。

func lineSpacing(CGFloat) -> View

Sets the amount of space between lines of text in the view. ビューの中のテキストの行間の空間の量を設定します。

func multilineTextAlignment(TextAlignment) -> View
func minimumScaleFactor(CGFloat) -> View

Sets the minimum amount that text scales down to fit the available space. テキストがその理由可能な空間に合うように縮小する最小限の量を設定します。

func truncationMode(Text.TruncationMode) -> View

Sets the truncation mode for lines of text that are too long to fit in the available space. 利用可能な空間に適合するには長すぎるテキストの行に対する切り詰めモードを設定します。

func allowsTightening(Bool) -> View

Sets whether text can compress the space between characters when necessary to fit text in a line. テキストを１行に収めるために必要な場合に、テキストが文字の間の空間を圧縮できるかどうかを設定します。

func textContentType(UITextContentType?) -> View

Sets the text content type for this view, which the system uses to offer suggestions while the user enters text on an iOS or tvOS device. テキスト内容型をこのビューに対して設定します、それはシステムが提案を申し出るために使います、ユーザがテキストをiOSまたはtvOSデバイス上で入力する間に。

func textContentType(WKTextContentType?) -> View

Sets the text content type for this view, which the system uses to offer suggestions while the user enters text on a watchOS device. テキスト内容型をこのビューに対して設定します、それはシステムが提案を申し出るために使います、ユーザがテキストをwatchOSデバイス上で入力する間に。

func flipsForRightToLeftLayoutDirection(Bool) -> View

Sets whether the view flips its contents horizontally when the layout direction is right-to-left. ビューがそれの内容を水平にフリップするかどうかを設定します、レイアウト方向が右から左である場合に。

func autocapitalization(UITextAutocapitalizationType) -> View

Sets whether to apply auto-capitalization to this view. 自動キャピタライゼーションをこのビューに適用すべきかどうかを設定します。

func disableAutocorrection(Bool?) -> View

Sets whether to disable autocorrection for this view. 自動修正をこのビューに対して無効にするべきかどうかを設定します。

Attaches a gesture to the view with a lower precedence than gestures defined by the view. ジェスチャをビューに添付します、ビューによって定義されるジェスチャより低い優先度で。

func highPriorityGesture<T>(T, including: GestureMask) -> View

Attaches a gesture to the view with a higher precedence than gestures defined by the view. ジェスチャをビューに添付します、ビューによって定義されるジェスチャより高い優先度で。

func simultaneousGesture<T>(T, including: GestureMask) -> View

Attaches a gesture to the view to process simultaneously with gestures defined by the view. ジェスチャをビューに添付します、ビューによって定義されるジェスチャと同期的に処理することになります。

func digitalCrownRotation<V>(Binding<V>) -> View

func digitalCrownRotation<V>(Binding<V>, from: V, through: V, by: V.Stride?, sensitivity: DigitalCrownRotationalSensitivity, isContinuous: Bool, isHapticFeedbackEnabled: Bool) -> View

func transaction((inout Transaction) -> Void) -> View

Handling View Events ビューイベントを取り扱う

func onAppear(perform: (() -> Void)?) -> View

Adds an action to perform when the view appears. ビューが姿を現わす時に実行するアクションを加えます。

func onDisappear(perform: (() -> Void)?) -> View

Adds an action to perform when the view disappears. ビューが姿を消す時に実行するアクションを加えます。

func onCutCommand(perform: (() -> [NSItemProvider])?) -> View

Adds an action to perform in response to the system’s Cut command. システムのもつ「カット」コマンドに答えて実行するアクションを加えます。

func onCopyCommand(perform: (() -> [NSItemProvider])?) -> View

Adds an action to perform in response to the system’s Copy command. システムのもつ「コピー」コマンドに答えて実行するアクションを加えます。

func onPasteCommand(of: [String], perform: ([NSItemProvider]) -> Void) -> View

Adds an action to perform in response to the system’s Paste command. システムのもつ「ペースト」コマンドに答えて実行するアクションを加えます。

func onPasteCommand<Payload>(of: [String], validator: ([NSItemProvider]) -> Payload?, perform: (Payload) -> Void) -> View

Adds an action to perform in response to the system’s Paste command with items that you validate. システムのもつ「ペースト」コマンドに答えて、あなたが検証する項目とともに実行するアクションを加えます。

func onDeleteCommand(perform: (() -> Void)?) -> View

Adds an action to perform in response to the system’s Delete command. システムのもつ「削除」コマンドに答えて実行するアクションを加えます。

func onMoveCommand(perform: ((MoveCommandDirection) -> Void)?) -> View

Adds an action to perform in response to a move command, like when the user presses an arrow key on a Mac keyboard, or taps the edge of the Siri Remote when controlling an Apple TV. ユーザが矢印キーをMacキーボード上で押す時のような、またはApple TVを制御している時にSir Remoteの縁をタップするような、移動コマンドに答えて実行するアクションを加えます。

func onExitCommand(perform: (() -> Void)?) -> View

Sets up an action that triggers in response to receiving the exit command while the view has focus. ビューがフォーカスを持つ間に退出命令を受け取ることに答えて引き起こされるアクションを準備します。

func onPlayPauseCommand(perform: (() -> Void)?) -> View

Adds an action to perform in response to the system’s Play/Pause command. システムのもつ「再生／一時停止」コマンドに答えて実行するアクションを加えます。

func onCommand(Selector, perform: (() -> Void)?) -> View

Adds an action to perform in response to the given selector. 与えられたセレクタに答えて実行するアクションを加えます。

func deleteDisabled(Bool) -> View

Adds a condition for whether the view’s view hierarchy is deletable. ビューのもつビュー階層が削除可能であるかどうかに対するある条件を加えます。

func moveDisabled(Bool) -> View

Adds a condition for whether the view's view hierarchy is movable. ビューのもつビュー階層が移動可能であるかどうかに対するある条件を加えます。

func onDrop(of: [String], isTargeted: Binding<Bool>?, perform: ([NSItemProvider]) -> Bool) -> View
