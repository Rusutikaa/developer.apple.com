View Layout and Presentation ビューのレイアウトと提示

Combine views in stacks, generate groups and lists of views dynamically, and define view presentations and hierarchy. ビューをスタックにおいて組み合わせて、ビューのグループとリストを動的に生成して、そしてビューの提示と階層を定義してください。

Framework フレームワーク

  • SwiftUI

On This Page この項には

  • Overview 
  • Topics 
  • See Also 

Overview 概要

Use stacks and lists to lay out the views of your user interface. You can combine static views with views that are dynamically generated from collections of data. All container views update and adjust the positions of their children in response to changes in content or interface dimensions. スタックとリストを使って、あなたのユーザインターフェイスのビューそれらを配置してください。あなたは、静的ビューを、動的にデータのコレクションから生成されるビューと結合できます。全てのコンテナビューは、内容またはインターフェイス次元それらにおける変化に答えて、それらの子の位置を更新および調節します。

Topics 話題

Essentials 要点

Building Lists and Navigation リストとナビゲーションを構築する
With the basic landmark detail view set up, you need to provide a way for users to see the full list of landmarks, and to view the details about each location. 基本ランドマーク詳細ビュー設定で、あなたはユーザにある方法を提供する必要があります、ランドマークの完全なリストを見るために、そしてそれぞれの場所についての詳細を眺めるために。
Composing Complex Interfaces 複雑なインターフェイスを構成する
The home screen for Landmarks shows a scrolling list of categories, with horizontally scrolling landmarks within each category. As you build this primary navigation, you’ll explore how composed views can adapt to different device sizes and orientations. Landmarksに対するホーム画面は、いくつかのカテゴリからなるあるスクロールリストを示します、各カテゴリ内で水平にスクロールするランドマークを備えます。あなたがこの主ナビゲーションをビルドするとき、あなたは、構成されたビューが異なるデバイスサイズおよび方向にどのように順応するか探査するでしょう。

Stacks

struct HStack
A view that arranges its children in a horizontal line. あるビュー、それは、それの子らをある水平線に整列します。
struct VStack
A view that arranges its children in a vertical line. あるビュー、それは、それの子らをある垂直線に整列します。
struct ZStack
A view that overlays its children, aligning them in both axes. あるビュー、それは、それの子らを上に重ねます、それらを両方の軸で整列しています。
struct LazyHStack
A view that arranges its children in a line that grows horizontally, creating items only as needed.
struct LazyVStack
A view that arranges its children in a line that grows vertically, creating items only as needed.

Grids

struct LazyHGrid
A container view that arranges its child views in a grid that grows horizontally, creating items only as needed.
struct LazyVGrid
A container view that arranges its child views in a grid that grows vertically, creating items only as needed.
struct GridItem
A description of a single grid item, such as a row or a column.

Lists and Scroll Views リストおよびスクロールビュー

struct List
A container that presents rows of data arranged in a single column. あるコンテナ、それは単一の縦列の中に配置されるデータの横列それらを提示します。
struct ForEach
A structure that computes views on demand from an underlying collection of of identified data. ある構造体、それはビューそれらを要請に応じてある識別されたデータの基礎をなすコレクションから計算します。
struct ScrollView
A scrollable view. スクロール可能なビュー。
struct ScrollViewReader
A view whose child is defined as a function of a ScrollViewProxy targeting the scrollable views within the child.
struct ScrollViewProxy
A proxy value allowing the scrollable views within a view hierarchy to be scrolled programmatically.
protocol DynamicViewContent
A type of view that generates views from an underlying collection of data. ビューのある型、それはある基礎をなすデータのコレクションからビューそれらを生成するものです。
enum Axis
The horizontal or vertical dimension in a 2D coordinate system. ある2D座標システムにおける水平または垂直次元。

Container Views コンテナビュー

struct Form
A container for grouping controls used for data entry, such as in settings or inspectors. たとえば設定または調査において、データ登録項目に対して使われるコントロールをグループにするためのコンテナ。
struct Group
An affordance for grouping view content. ビュー内容をまとめることに対するアフォーダンス。
struct GroupBox
A stylized view with an optional label that is associated with a logical grouping of content. 内容の論理グループと結びつけられる随意のラベルでスタイル指定されたビュー。
struct Section
An affordance for creating hierarchical view content. 階層ビュー内容を作成することに対するアフォーダンス。

Hierarchical Views 階層ビュー

struct OutlineGroup
A structure that computes views and disclosure groups on demand from an underlying collection of tree-structured, identified data. ある構造体、それはビューおよび開示グループを要求に応じてツリー構造の、識別可能なデータからなるある基礎をなすコレクションから計算します。
struct DisclosureGroup
A view that shows or hides another content view, based on the state of a disclosure control. あるビュー、それは別のコンテンツビューを、ある開示コントロールの状態に基づき表示するまたは隠します。

Spacers and Dividers 間隔子と分割子

struct Spacer
A flexible space that expands along the major axis of its containing stack layout, or on both axes if not contained in a stack. ある柔軟な空き、それはそれを収容しているスタックレイアウトの長軸に沿って、またはスタックの中に含まれないならば両方の軸上で拡張します。
struct Divider
A visual element that can be used to separate other content. ある視覚的要素、それは他の内容を隔てるために使われることができます。

Architectural Views 建築上のビュー

struct NavigationView
A view for presenting a stack of views representing a visible path in a navigation hierarchy. ナビゲーション階層において、ある可視パスを表しているビューいくつかからなるあるスタックを提示するためのあるビュー。
struct TabView
A view that switches between multiple child views using interactive user interface elements. あるビュー、それは複数の子ビューの間を切り替えます、対話式ユーザインターフェイス要素を使います。
struct HSplitView
A layout container that arranges its children in a horizontal line and allows the user to resize them using dividers placed between them. あるレイアウトコンテナ、それは、それの子らをある水平線に整列します、そしてそれらの間に置かれる仕切りを使ってユーザにそれらの大きさ変更を可能します。
struct VSplitView
A layout container that arranges its children in a vertical line and allows the user to resize them using dividers placed between them. あるレイアウトコンテナ、それは、それの子らをある垂直線に整列します、そしてそれらの間に置かれる仕切りを使ってユーザにそれらの大きさ変更を可能します。

Presentations 提示

struct Alert
A representation for an alert presentation.
struct ActionSheet
A storage type for an action sheet presentation. アクションシート提示のためのストレージ型。

Conditionally Visible Items 条件付きで見ることができる項目

struct EmptyView
struct EquatableView
A view type that compares itself against its previous value and prevents its child updating if its new value is the same as its old value. あるビュー型、それはそれ自身をそれの前の値と比較して、それの新しい値がそれの古い値と同じならば、それの子を更新することを防ぎます。

Infrequently Used Views まれに使われるビュー

struct AnyView
A type-erased view.
struct TupleView
A View created from a swift tuple of View values. View値それらからなるあるswiftタプルから作成されるあるView。

See Also 参照

User Interface ユーザインターフェイス

Views and Controls ビューとコントロール
Present your content onscreen and handle user interactions. 画面上にあなたのコンテンツを提示します、そしてユーザ相互作用を取り扱います。
Drawing and Animation 描画とアニメーション
Enhance your views with colors, shapes, and shadows, and customize animated transitions between view states. あなたのビューを色、輪郭、そして影で強調してください、そしてビューの状態間での転換アニメーションをカスタマイズしてください。
Framework Integration フレームワーク統合
Integrate SwiftUI views into existing apps, and embed AppKit, UIKit, and WatchKit views and controllers into SwiftUI view hierarchies. SwiftUIビューを既存のアプリへと、そして埋め込みAppKit、UIKit、そしてWatchKitビューとコントローラをSwiftUIビュー階層へと統合しましょう。
