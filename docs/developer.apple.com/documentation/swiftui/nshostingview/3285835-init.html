Initializer

init(frame:)

Initializes and returns a newly allocated AppKit view with a specified frame rectangle. 新しくアロケートされたAppKitビューをある指定されたフレーム矩形で初期化して返します。

SDKs

  • macOS 10.15+
  • Xcode 11.0+

Framework フレームワーク

  • SwiftUI

On This Page この項には

  • Declaration 宣言
  • Parameters パラメータ
  • Return Value 戻り値
  • Discussion 議論
  • See Also 参照

Declaration 宣言

Parameters パラメータ

frameRect

The frame rectangle for the new view object. 新しいビューオブジェクトに対するフレーム矩形。

Return Value 戻り値

An initialized NSHostingView object or nil if the object couldn't be created. 初期化されたNSHostingViewオブジェクトまたはオブジェクトが作成されることができなかったならばnil。

Discussion 議論

Use the init(rootView:) method to create your hosting view instead. init(rootView:)メソッドを使って、あなたのホスティングビューを代わりに作成してください。

See Also 参照

Creating a Hosting View ホスティングビューを作成する

init(rootView: Content)

Creates a hosting view object that wraps the specified SwiftUI view. あるホスティングビューオブジェクトを作成します、それは指定されたSwiftUIビューをラップします。

init?(coder: NSCoder)

Creates a hosting view object from the contents of the specified archive. ホスティングビューオブジェクトをこの指定されたアーカイブの内容から作成します。
