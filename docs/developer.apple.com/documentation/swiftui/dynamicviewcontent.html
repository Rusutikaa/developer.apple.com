Protocol

DynamicViewContent

A type of view that generates views from an underlying collection of data. ビューのある型、それはある基礎をなすデータのコレクションからビューそれらを生成するものです。

SDKs

  • iOS 13.0+
  • macOS 10.15+
  • Mac Catalyst 13.0+
  • tvOS 13.0+
  • watchOS 6.0+
  • Xcode 11.0+

Framework フレームワーク

  • SwiftUI

Declaration 宣言

Topics 話題

Managing the Data データを管理する

var data: Self.Data

The collection of underlying data. 基礎をなすデータのコレクション。

Required. 必須。

associatedtype Data

The type of the underlying collection of data. 基礎をなすデータのコレクションの型。

Required. 必須。

Responding to Updates 更新に応答する

func onDelete(perform: Optional<(IndexSet) -> Void>) -> DynamicViewContent

Sets the deletion action for the dynamic view. 動的ビューに対する削除アクションを設定します。

func onInsert(of: [String], perform: (Int, [NSItemProvider]) -> Void) -> DynamicViewContent

Sets the insert action for the dynamic view. 動的ビューに対して挿入アクションを設定します。

func onMove(perform: Optional<(IndexSet, Int) -> Void>) -> DynamicViewContent

Sets the move action for the dynamic view. 動的ビューに対する移動アクションを設定します。

Relationships 関係

Inherits From 継承元

View

Conforming Types これらの型が準拠

  • ForEach
    Conforms when Content conforms to View. ContentがViewに準拠する時に準拠します。
  • ModifiedContent
    Conforms when Content conforms to DynamicViewContent and Modifier conforms to ViewModifier. ContentがDynamicViewContentに準拠するそしてModifierがViewModifierに準拠する時に準拠します。

struct List

struct ForEach

The collection of underlying identified data that SwiftUI uses to create views dynamically. 基礎をなす識別されたデータのコレクション、それはSwiftUIが使用してビューを動的に作成するものです。

struct ScrollView

A scrollable view. スクロール可能なビュー。
