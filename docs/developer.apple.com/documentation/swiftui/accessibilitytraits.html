AccessibilityTraits

A set of accessibility traits that describe how an element behaves. 一揃いのアクセシビリティ特徴（トレイト）、それはどのように要素が振舞うかを記述します。

SDKs

  • iOS 13.0+
  • macOS 10.15+
  • Mac Catalyst 13.0+
  • tvOS 13.0+
  • watchOS 6.0+
  • Xcode 11.0+
