Drawing and Animation 描画とアニメーション

Enhance your views with colors, shapes, and shadows, and customize animated transitions between view states. あなたのビューを色、輪郭、そして影で強調してください、そしてビューの状態間での転換アニメーションをカスタマイズしてください。

Framework フレームワーク

  • SwiftUI

  • Overview 概要
  • Topics 話題
  • See Also 参照

Overview 概要

Use drawing tools to combine shapes or define custom paths for shapes of your own design. Apply styles from environment-aware colors to rich gradients to the foreground, background, and outline of your shapes. 描画ツールを使って、形状を結合してください、またはあなた独自のデザインの形状のためにあつらえのパスを定義してください。環境認知色から豊かな勾配までのさまざまなスタイルをあなたの形状の前面、背面、そして輪郭に適用してください。

Topics 話題

Essentials 要点

Drawing Paths and Shapes パスと形状を描画する

Users receive a badge whenever they visit a landmark in their list. Of course, for a user to receive a badge, you’ll need to create one. This tutorial takes you through the process of creating a badge by combining paths and shapes, which you then overlay with another shape that represents the location. ユーザは、彼らが彼らのリストにおいてランドマークを訪問する時はいつでも、バッジを受け取ります。もちろん、バッジを受け取るユーザに対して、あなたはそれを作成する必要があるでしょう。このチュートリアルは、あなたにバッジ作成の過程をパスと形状を組み合わせることで手取り足取り指導します、それはあなたがそれから場所を表す別の形状をかぶせるものです。

Building Custom Views in SwiftUI あつらえのビューをSwiftUIにおいて構築する

Create a custom view with data driven transitions and animations in SwiftUI. あつらえのビューをデータ駆動の転換およびアニメーションでSwiftUIにおいて作成します。

protocol Shape

A 2D shape that you can use when drawing a view. ある2D形状、それはあなたがビューを描画する時に使用できるものです。

Animation アニメーション

Animating Views and Transitions ビューと遷移をアニメーション化する

When using SwiftUI, you can individually animate changes to views, or to a view’s state, no matter where the effects are. SwiftUI handles all the complexity of these combined, overlapping, and interruptible animations for you. SwiftUIを使う場合、あなたはビューに対する、またはビューのもつ状態に対する変化を個々にアニメーションにできます、その効果がどこにあるとしても。SwiftUIは、組み合わされた、オーバーラップしている、そして中断可能なアニメーションの複雑さの全てをあなたの代わりに取り扱います。

struct Animation
protocol Animatable

A type for animating views. アニメーションビューに対する型。

protocol AnimatableModifier

A modifier that can create another modifier with animation. ある修飾子、それはアニメーションを備える別の修飾子を作成できます。

func withAnimation<Result>(Animation?, () -> Result) -> Result

Returns the result of recomputing the view’s body with the provided animation. ビューのもつボディをこの提供されたアニメーションで再計算する結果を返します。

struct AnimatablePair

A pair of animatable values, which is itself animatable. 一対のアニメーション値、それはそれ自身アニメーション可能です。

struct EmptyAnimatableData

An empty type for animatable data. アニメーション可能データに対するある空の型。

struct AnyTransition

A type-erased transition. 型消去遷移。

Shapes

struct Rectangle

A rectangular shape aligned inside the frame of the view containing it. ある長方形、それを含んでいるビューの枠内に整列されます。

enum Edge

An enumeration to indicate one edge of a rectangle. ある矩形の１つの縁を指し示すある列挙。

struct RoundedRectangle

A rectangular shape with rounded corners, aligned inside the frame of the view containing it. 角を丸められたある長方形、それを含んでいるビューの枠内に整列されます。

struct Circle

A circle centered on the frame of the view containing it. ある円、それを含んでいるビューの枠内の中心に置かれます。

struct Ellipse

An ellipse aligned inside the frame of the view containing it. ある楕円、それを含んでいるビューの枠内に整列されます。

struct Capsule

A capsule shape aligned inside the frame of the view containing it. それを含んでいるビューのフレーム内部に配置されるカプセル形状。

struct Path

The outline of a 2D shape. 2D形状の輪郭。

Transformed Shapes 変形された形状

protocol InsettableShape

A shape type that is able to inset itself to produce another shape. ある形状型、それはそれ自身にインセット（縁差込）をすることで別の形状を生み出せます。

struct ScaledShape

A shape with a scale transform applied to it. それへと適用される拡大縮小変形した状態でのある形状。

struct RotatedShape

A shape with a rotation transform applied to it. それへと適用される回転変形した状態でのある形状。

struct OffsetShape

A shape with a translation offset transform applied to it. それへと適用される転換オフセット変形した状態でのある形状。

struct TransformedShape

A shape with an affine transform applied to it. それへと適用されるアフィン変換した状態でのある形状。

Paints, Styles, and Gradients ペイント、スタイル、そして勾配

struct Color

An environment-dependent color. 環境依存色。

struct ImagePaint

A paint type that repeats an image over the infinite plane. あるペイント型、それはある画像を無限面にわたって繰り返します。

struct Gradient

A color gradient represented as an array of color stops, each having a parametric location value. 色経由それらからなるある配列として表されるある色勾配、それぞれはパラメータ的位置値を持ちます。

struct LinearGradient

A linear gradient. 線形勾配。

struct AngularGradient

An angular gradient. 角勾配（角度によるグラデーション）。

struct RadialGradient

A radial gradient. 放射状の色勾配。

struct ForegroundStyle
struct FillStyle

A style for rasterizing vector shapes. ベクター形状のラスタライズに対するスタイル。

protocol ShapeStyle

A way to turn a shape into a view. ある形状をビューへと変える方法。

enum RoundedCornerStyle

Defines the shape of a rounded rectangle’s corners. 丸められた矩形のもつ角の形状を定義します。

struct SelectionShapeStyle

Geometry 幾何学

struct GeometryProxy

A proxy for access to the size and coordinate space (for anchor resolution) of the container view. コンテナビューの（アンカー解決のための）大きさと座標空間にアクセスするためのプロキシ。

struct GeometryReader

A container view that defines its content as a function of its own size and coordinate space. あるアンカービュー、それはそれの内容をそれ独自の大きさと座標空間の相関関係として定義します。

protocol GeometryEffect

An effect that changes the visual appearance of a view, largely without changing its ancestors or descendents. ある効果、それはビューの視覚的外観を変更します、大部分はそれのアンカーまたは子孫それらを変更することなしに。

struct Angle

A geometric angle whose value you access in either radians or degrees. 幾何学的角度、それの値にあなたはラジアンまたは度のどちらかでアクセスします。

struct Anchor

An opaque value derived from an anchor source and a particular view. アンカーソースおよび特定のビューから引き出されるある不透明値。

struct UnitPoint
enum CoordinateSpace
struct ProjectionTransform
protocol VectorArithmetic

A type that can serve as the animatable data of an animatable type. ある型、それはアニメーション可能型のアニメーション可能データとしての用をなすものです。
