EditMode

The mode of a view indicating whether the user can edit its content. ユーザがそれの内容を編集可能かどうかを指し示しているあるビューのモード。

SDKs

  • iOS 13.0+
  • Mac Catalyst 13.0+

Topics 話題

Checking for Editing Mode

var isEditing: Bool

Indicates whether a view is being edited. ビューが編集されているかどうかを指し示します。

Modes

case active

The view content can be edited. ビュー内容は、編集されことができます。

case inactive

The view content cannot be edited. ビュー内容は、編集されことができません。

case transient

The view is in a temporary edit mode. ビューは、一時的編集モードにあります。
