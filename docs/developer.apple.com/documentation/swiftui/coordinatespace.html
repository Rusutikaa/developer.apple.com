Geometry 幾何学

struct GeometryProxy

A proxy for access to the size and coordinate space (for anchor resolution) of the container view. コンテナビューの（アンカー解決のための）大きさと座標空間にアクセスするためのプロキシ。

struct GeometryReader

A container view that defines its content as a function of its own size and coordinate space. あるアンカービュー、それはそれの内容をそれ独自の大きさと座標空間の相関関係として定義します。

protocol GeometryEffect

An effect that changes the visual appearance of a view, largely without changing its ancestors or descendents. ある効果、それはビューの視覚的外観を変更します、大部分はそれのアンカーまたは子孫それらを変更することなしに。

struct Angle

A geometric angle whose value you access in either radians or degrees. 幾何学的角度、それの値にあなたはラジアンまたは度のどちらかでアクセスします。

struct Anchor

An opaque value derived from an anchor source and a particular view. アンカーソースおよび特定のビューから引き出されるある不透明値。

struct UnitPoint
struct ProjectionTransform
protocol VectorArithmetic

A type that can serve as the animatable data of an animatable type. ある型、それはアニメーション可能型のアニメーション可能データとしての用をなすものです。
