Generic Instance Method 総称体インスタンスメソッド

digitalCrownRotation(_:)

Tracks Digital Crown rotations by updating the specified binding. 「デジタルクラウン」の回転をこの指定されたバインディングを更新することによって追跡します。

SDKs

  • watchOS 6.0+
  • Xcode 11.0+

Framework フレームワーク

  • SwiftUI

On This Page この項には

  • Declaration 宣言
  • Parameters パラメータ
  • See Also 参照

Declaration 宣言

Parameters パラメータ

binding

A value that updates as the user rotates the Digital Crown. The implicit range is (-infinity, +infinity). ある値、それはユーザが「デジタルクラウン」を回すにつれて更新されます。暗黙的な範囲は、(-infinity, +infinity)です。
