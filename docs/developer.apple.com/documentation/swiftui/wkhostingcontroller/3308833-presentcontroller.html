Instance Method インスタンスメソッド

presentController(withNamesAndContexts:)

Presents one or more new interface controllers modally using a page-based arrangement. １つ以上の新しいインターフェイスコントローラをモーダルにページ基盤整列を使って提示します。

SDKs

  • watchOS 6.0+
  • Xcode 11.0+

Framework フレームワーク

  • SwiftUI

  • Declaration 宣言
  • Parameters パラメータ
  • Discussion 議論
  • See Also 参照

Declaration 宣言

Parameters パラメータ

namesAndContexts

An array of tuples, each of which contains the following elements: いくつかのタプルからなるある配列、そのそれぞれが以下の要素を含みます：

name

The names of the interface controller you want to display. In storyboards, specify the name of your view controller in the Identifier property of the Attributes inspector. あなたが表示したいインターフェイスコントローラの名前。ストーリーボードにおいて、あなたのビューコントローラの名前をAttributesインスペクタのIdentifierプロパティにおいて指定してください。

context

The objects you want to pass to the interface controllers. Use this parameter to communicate important information to each new interface controller. For example, use it to provide data or relevant state information. あなたがインターフェイスコントローラに渡したいオブジェクト。このパラメータを使って、重要な情報を新しいインターフェイスコントローラのそれぞれに伝達してください。例えば、それを使ってデータまたは関連がある状態情報を提供してください。

Discussion 議論

Use this method to present new WKInterfaceController or WKHostingController objects modally onscreen. The new interface slides up from the bottom of the screen and completely covers the contents of the current interface controller. The interface displays the first interface controller listed in the namesAndContexts array initially. The user can navigate to other interface controllers by swiping horizontally. このメソッドを使って、新しいWKInterfaceControllerまたはWKHostingControllerオブジェクトをモーダルに画面上に提示してください。新しいインターフェイスは、画面の下からせり上がります、そして完全に現在のインターフェイスコントローラの内容を覆います。インターフェイスは、初めに、namesAndContexts配列においてリストされる最初のインターフェイスコントローラを表示します。ユーザは、他のインターフェイスコントローラになんとか進むことが水平にスワイプすることで可能です。

The title of the modal interface is set to Cancel by default, but you can change the title using the setTitle(_:) method. Call the dismiss() method to dismiss the interface programmatically. The user can also dismiss the interface by tapping the title. モーダルインターフェイスのタイトルは、Cancelに初期状態では設定されます、しかしあなたはタイトルをsetTitle(_:)メソッドを使って変更できます。dismiss()メソッドを呼び出して、インターフェイスをプログラムに基づいて退出させてください。ユーザは、またインターフェイスを退出させることがタイトルをタップすることで可能です。

Always call this method from your WatchKit extension's main thread. 常にこのメソッドをあなたのWatchKit拡張のもつメインスレッドから呼び出してください。
