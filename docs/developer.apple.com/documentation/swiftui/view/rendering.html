Rendering レンダリング

Affect the way the system draws a view, for example by scaling or masking a view, or by applying graphical effects. システムがビューを描画する方法に影響を与えます、例えばあるビューを拡大縮小またはマスクすることによって、またはグラフィカル効果を適用することによって。

Framework フレームワーク

  • SwiftUI

Topics 話題

Masking and Clipping Views ビューを隠すおよび切り抜く

func clipped(antialiased: Bool) -> View

Clips the view to its bounding rectangular frame. ビューをそれの境界矩形枠に切り抜きます。

func clipShape<S>(S, style: FillStyle) -> View

Sets a clipping shape for the view. ある切り抜き形状をビューに設定します。

func cornerRadius(CGFloat, antialiased: Bool) -> View

Clips the view to its bounding frame, with the specified corner radius. ビューをそれの境界枠に切り抜きます、この指定された角丸めで。

func mask<Mask>(Mask) -> View

Masks this view using the alpha channel of the given view. このビューをこの与えられたビューのアルファチャンネルを使って隠します。

Scaling Views ビューを縮尺変更する

func scaledToFill() -> View

Scales the view to fill its parent. ビューを拡大縮小してそれの親をいっぱいに満たします。

func scaledToFit() -> View

Scales the view to fit its parent. ビューを縮尺変更してそれの親にぴったり合わせます。

func scaleEffect(CGFloat, anchor: UnitPoint) -> View

Scales this view’s rendered output by the given amount in both the horizontal and vertical directions, relative to an anchor point. このビューのもつ描画された出力をこの与えられた量によって水平と垂直方向の両方で、あるアンカーポイントに関して、縮尺変更します。

func scaleEffect(CGSize, anchor: UnitPoint) -> View

Scales this view’s rendered output by the given vertical and horizontal size amounts, relative to an anchor point. このビューのもつ描画された出力をこの与えられた水平と垂直の大きさ量によって、あるアンカーポイントに関して、縮尺変更します。

func scaleEffect(x: CGFloat, y: CGFloat, anchor: UnitPoint) -> View

Scales this view’s rendered output by the given horizontal and vertical amounts, relative to an anchor point. このビューのもつ描画された出力をこの与えられた水平と垂直の量によって、あるアンカーポイントに関して、縮尺変更します。

func aspectRatio(CGFloat?, contentMode: ContentMode) -> View

Constrains the view’s dimensions to the specified aspect ratio. ビューのもつ寸法をこの指定されたアスペクト比に強制します。

func aspectRatio(CGSize, contentMode: ContentMode) -> View

Constrains the view’s dimensions to the aspect ratio of the specified size. ビューのもつ寸法をこの指定された大きさのアスペクト比に強制します。

enum ContentMode

Constants that define how a view's content fills the available space. どのようにビューのもつ内容がその利用可能な空間を満たすかを定義する定数。

func imageScale(Image.Scale) -> View

Sets the scale of images inside the view. ビュー内の画像の縮尺を設定します。

Rotating and Transforming Views ビューを回転するおよび変形する

func rotationEffect(Angle, anchor: UnitPoint) -> View

Rotates this view’s rendered output around the specified point. このビューのもつ描画された出力をこの指定された点を中心として回転します。

func rotation3DEffect(Angle, axis: (x: CGFloat, y: CGFloat, z: CGFloat), anchor: UnitPoint, anchorZ: CGFloat, perspective: CGFloat) -> View

Rotates this view’s rendered output in three dimensions around the given axis of rotation. このビューのもつ描画された出力を３次元においてこの指定された回転軸を中心として回転します。

func projectionEffect(ProjectionTransform) -> View

Applies a projection transformation to this view’s rendered output. 射影変換をこのビューのもつ描画された出力に適用します。

func transformEffect(CGAffineTransform) -> View

Applies an affine transformation to this view’s rendered output. アフィン変換をこのビューのもつ描画された出力に適用します。

Applying Graphical Effects to a View グラフィカル効果をビューに適用する

func blur(radius: CGFloat, opaque: Bool) -> View

Applies a Gaussian blur to the view. ガウスぼかしをビューに適用します。

func opacity(Double) -> View

Sets the transparency of the view. ビューの透明度を設定します。

func brightness(Double) -> View

Brightens the view by the specified amount. ビューをこの指定された量だけ明るくします。

func contrast(Double) -> View

Sets the contrast and separation between similar colors in the view. ビューの中の類似した色の間のコントラストと分解を設定します。

func colorInvert() -> View

Inverts the colors in the view. ビューの中の色を反転します。

func colorMultiply(Color) -> View

Adds a color multiplication effect to the view. ある色乗算効果をビューに加えます。

func saturation(Double) -> View

Adjusts the color saturation of the view. ビューの色彩度を補正します。

func grayscale(Double) -> View

Adds a grayscale effect to the view. グレイスケール効果をビューに加えます。

func hueRotation(Angle) -> View

Applies a hue rotation effect to the view. 色相回転効果をビューに適用します。

func luminanceToAlpha() -> View

Adds a luminance to alpha effect to the view. 輝度をビューへのアルファ効果に加えます。

func shadow(color: Color, radius: CGFloat, x: CGFloat, y: CGFloat) -> View

Adds a shadow to the view. 影をビューに加えます。

Compositing Views ビューを合成する

func blendMode(BlendMode) -> View

Sets the blend mode for compositing the view with overlapping views. 混合モードを、重なり合っているビューをもつビューの合成に対して設定します。

enum BlendMode

Modes for compositing a view with overlapping content. あるビューをオーバーラッブ内容と合成するモード。

func compositingGroup() -> View

Wraps the view in a compositing group. ビューをある合成グループの中に包みます。

func drawingGroup(opaque: Bool, colorMode: ColorRenderingMode) -> View

Composites the view’s contents into an offscreen image before final display. ビューのもつ内容をある画面外の画像へと最終的表示の前に合成します。

enum ColorRenderingMode

The working color space for color-compositing operations. 色合成演算に対する作業色空間。

Adding Animations to a View アニメーションをビューに加える

func animation(Animation?) -> View

Applies the given animation to all animatable values within the view. 与えられたアニメーションをビュー内の全てのアニメーション可能な値に適用します。

func animation<V>(Animation?, value: V) -> View

Applies the given animation to the view when the specified value changes. 与えられたアニメーションをビューに適用します、指定された値が変化する時に。

func transition(AnyTransition) -> View

Associates a transition with the view. ある遷移をビューと結びつけます。

See Also 参照

Configuring Views ビューを構成設定する

Layout レイアウト

Tell a view how to arrange itself within a view hierarchy by adjusting its size, position, alignment, padding, and so on. ビューにどのようにそれ自身をビュー階層内で整えるか伝えます、それの大きさ、位置、整列、詰め、その他を調整することによって。

Accessibility アクセシビリティ

Make your views accessible to a broad range of users, including people with disabilities. あなたのビューを広範囲のユーザにアクセス可能にします、身体障害をもつ人々を含めて。

Styling スタイリング

Indicate how to style the text, controls, and other content contained in a view. どのようにビューの中に含まれるテキスト、コントロール、および他の内容をスタイル指定するかを指し示します。
