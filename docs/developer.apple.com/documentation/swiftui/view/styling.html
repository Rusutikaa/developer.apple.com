Styling

Indicate how to style the text, controls, and other content contained in a view.

Framework フレームワーク

  • SwiftUI

On This Page この項には

  • Topics 話題
  • See Also 参照

Topics 話題

Setting the Border of a View ビューの境界線を設定する

func border<S>(S, width: CGFloat) -> View

Adds a border to the view with the specified style and width. 境界線をビューにこの指定されたスタイルと幅で加えます。

Setting View Colors ビューカラーを設定する

func foregroundColor(Color?) -> View

Sets the color that the view uses for foreground elements. 色を設定します、それはビューが前面要素に使います。

func accentColor(Color?) -> View

Sets the accent color for the view and the views it contains. アクセントカラーをそのビューとそれが含むビューそれらに対して設定します。

Adopting View Color Schemes ビューのカラースキームを採用する

func colorScheme(ColorScheme) -> View

Sets the view’s color scheme. ビューのもつカラースキームを設定します。

func preferredColorScheme(ColorScheme?) -> View

Sets the preferred color scheme for this view. 優先カラースキームをこのビューに対して設定します。

enum ColorScheme

The possible types of color schemes, like Dark Mode. カラースキームの可能な型、Dark Modeのような。

struct PreferredColorSchemeKey

A key for specifying the preferred color scheme. 望ましいカラースキームを指定するためのあるキー。

Adjusting Text in a View テキストをビューに合わせる

func keyboardType(UIKeyboardType) -> View

Sets the keyboard type for this view. キーボード型をこのビューに対して設定します。

func font(Font?) -> View

Sets the default font for text in the view. 省略時のフォントをビューの中のテキストに対して設定します。

func lineLimit(Int?) -> View

Sets the maximum number of lines that text can occupy in the view. テキストがビューにおいて占めることができる最大の行数を設定します。

func lineSpacing(CGFloat) -> View

Sets the amount of space between lines of text in the view. ビューの中のテキストの行間の空間の量を設定します。

func multilineTextAlignment(TextAlignment) -> View
func minimumScaleFactor(CGFloat) -> View

Sets the minimum amount that text scales down to fit the available space. テキストがその理由可能な空間に合うように縮小する最小限の量を設定します。

func truncationMode(Text.TruncationMode) -> View

Sets the truncation mode for lines of text that are too long to fit in the available space. 利用可能な空間に適合するには長すぎるテキストの行に対する切り詰めモードを設定します。

func allowsTightening(Bool) -> View

Sets whether text can compress the space between characters when necessary to fit text in a line. テキストを１行に収めるために必要な場合に、テキストが文字の間の空間を圧縮できるかどうかを設定します。

func textContentType(UITextContentType?) -> View

Sets the text content type for this view, which the system uses to offer suggestions while the user enters text on an iOS or tvOS device. テキスト内容型をこのビューに対して設定します、それはシステムが提案を申し出るために使います、ユーザがテキストをiOSまたはtvOSデバイス上で入力する間に。

func textContentType(WKTextContentType?) -> View

Sets the text content type for this view, which the system uses to offer suggestions while the user enters text on a watchOS device. テキスト内容型をこのビューに対して設定します、それはシステムが提案を申し出るために使います、ユーザがテキストをwatchOSデバイス上で入力する間に。

func flipsForRightToLeftLayoutDirection(Bool) -> View

Sets whether the view flips its contents horizontally when the layout direction is right-to-left. ビューがそれの内容を水平にフリップするかどうかを設定します、レイアウト方向が右から左である場合に。

func autocapitalization(UITextAutocapitalizationType) -> View

Sets whether to apply auto-capitalization to this view. 自動キャピタライゼーションをこのビューに適用すべきかどうかを設定します。

func disableAutocorrection(Bool?) -> View

Sets whether to disable autocorrection for this view. 自動修正をこのビューに対して無効にするべきかどうかを設定します。

Configuring Control Attributes コントロール属性を構成設定する

func labelsHidden() -> View

Hides the labels of any controls contained within this view. このビュー内に含まれるあらゆるコントロールのラベルを隠します。

func defaultWheelPickerItemHeight(CGFloat) -> View

Sets the default wheel-style picker item height. 省略時の環形式ピッカー項目の高さを設定します。

func horizontalRadioGroupLayout() -> View

Sets the style for radio group style pickers within the view’s environment to be radio buttons horizontally arranged inside of the layout. ラジオグループ形式ピッカーに対するスタイルをビューのもつ環境内で設定します、ラジオボタンが水平にそのレイアウト（割り付け）の内側で整頓されるようにします。

func controlSize(ControlSize) -> View

Sets the size class, like regular or small, for controls within this view. 大きさの種類を、標準または小型のように、このビュー内のコントロールに対して設定します。

enum ControlSize

The size classes, like regular or small, that you can apply to controls within a view. サイズクラス、レギュラーまたはスモールのような、それはあなたがビュー内のコントロールに適用できるものです。

Styling Specific View Types

func buttonStyle<S>(S) -> View

Sets the style for buttons within this view to a button style with a custom appearance and custom interaction behavior.

func buttonStyle<S>(S) -> View

Sets the style for buttons within this view to a button style with a custom appearance and standard interaction behavior.

func menuButtonStyle<S>(S) -> View

Sets the style for menu buttons within this view. このビュー内のメニューボタンそれらに対してスタイルを設定します。

func pickerStyle<S>(S) -> View

Sets the style for pickers within this view. このビュー内のピッカーそれらに対してスタイルを設定します。

func datePickerStyle<S>(S) -> View

Sets the style for date pickers within this view. このビュー内の日付ピッカーそれらに対してスタイルを設定します。

func textFieldStyle<S>(S) -> View

Sets the style for text fields within this view. このビュー内のテキストフィールドそれらに対してスタイルを設定します。

func toggleStyle<S>(S) -> View

Sets the style for toggles within this view. このビュー内のトグルそれらに対してスタイルを設定します。

func listStyle<S>(S) -> View

Sets the style for lists within this view. このビュー内のリストそれらに対してスタイルを設定します。

func navigationViewStyle<S>(S) -> View

Sets the style for navigation views within this view. このビュー内のナビゲーションビューそれらに対してスタイルを設定します。

View Styles

Find the built-in styles for different types of views.

Configuring a List View リストビューを構成設定する

func listRowInsets(EdgeInsets?) -> View

Sets the inset to be applied to the view when placed in a list. リスト内に置かれた時にビューに適用されることになるインセットを設定します。

func listRowBackground<V>(V?) -> View

Sets a view behind the view when placed in a list. リスト内に置かれた時にビューを後ろに設定します。

func listRowPlatterColor(Color?) -> View

Sets the color to apply to the system cell platter of the view when placed in a list. リスト内に置かれた時にビューのシステムセルプラッタに適用される色を設定します。

func tag<V>(V) -> View

Sets a tag for the view in order to differentiate it from a list of view options. ビューに対してタグを設定します、それをビューオプションのリストから区別するために。

Configuring Navigation and Status Bar Views ナビゲーションおよびステータスバービューを構成設定する

func navigationBarTitle(Text) -> View

Configures the title in the navigation bar for the view. ビューに対してナビゲーションバーにおけるタイトルを構成設定します。

func navigationBarTitle(Text, displayMode: NavigationBarItem.TitleDisplayMode) -> View

Configures the title of the navigation bar for the view. ビューに対してナビゲーションバーのタイトルを構成設定します。

func navigationBarTitle(LocalizedStringKey) -> View

Configures the title of this view's navigation bar with a localized string. このビューのもつナビゲーションバーのタイトルをローカライズされた文字列で構成設定します。

func navigationBarTitle<S>(S) -> View

Configures the title of this view’s navigation bar with a string. このビューのもつナビゲーションバーのタイトルをある文字列で構成設定します。

func navigationBarTitle(LocalizedStringKey, displayMode: NavigationBarItem.TitleDisplayMode) -> View

Configures the title of this view's navigation bar with a localized string and a display mode. このビューのもつナビゲーションバーのタイトルをローカライズされた文字列とある表示モードで構成設定します。

func navigationBarHidden(Bool) -> View

Hides the navigation bar for the view. ビューに対してナビゲーションバーを隠します。

func statusBar(hidden: Bool) -> View

Sets the visibility of the status bar for the view. ビューに対してステータスバーの可視性を設定します。

Configuring Navigation and Tab Bar Item Views ナビゲーションおよびタブバー項目ビューを構成設定する

func navigationBarBackButtonHidden(Bool) -> View

Hides the navigation bar back button for the view. ビューに対してナビゲーションバー戻るボタンを隠します。

func navigationBarItems<L>(leading: L) -> View

Configures the navigation bar items for the view. ビューに対してナビゲーションバー項目を構成設定します。

func navigationBarItems<L, T>(leading: L, trailing: T) -> View

Configures the navigation bar items for the view. ビューに対してナビゲーションバー項目を構成設定します。

func navigationBarItems<T>(trailing: T) -> View

Configures the navigation bar items for the view. ビューに対してナビゲーションバー項目を構成設定します。

struct NavigationBarItem

A configuration for a navigation bar that represents a view at the top of a navigation stack.

func tabItem<V>(() -> V) -> View

Sets the tab bar item associated with this view. このビューと結びつけられたタブバー項目を設定します。

Configuring Context Menu Views コンテキストメニュービューを構成設定する

func contextMenu<MenuItems>(ContextMenu<MenuItems>?) -> View

Attaches a contextual menu and its children to the view. あるコンテキストによるメニューとそれの子をビューに添付します。

func contextMenu<MenuItems>(menuItems: () -> MenuItems) -> View

Attaches a contextual menu to the view. あるコンテキストによるメニューをビューに添付します。

struct ContextMenu

A container for views that you present as a menu items in a contextual menu in response to the standard system gesture. あなたがメニュー項目それらとしてあるコンテキストメニューの中に標準システムジェスチャに応答して提示するビューそれらに対するあるコンテナ。

Configuring Touch Bar Views Touch Bar ビューを構成設定する

func touchBar<Content>(content: () -> Content) -> View

Sets the content that the Touch Bar displays. Touch Barが表示する内容を設定します。

func touchBar<Content>(TouchBar<Content>) -> View

Sets the touch bar and its content to be shown in the Touch Bar when applicable. タッチバーを設定してそれの内容をTouch Barにおいて表示されるようにします、適用可能な時に。

func touchBarItemPrincipal(Bool) -> View

Sets the view as having special significance to the TouchBar. 特別な意味をTouchBarに持つようにビューを設定します。

func touchBarCustomizationLabel(Text) -> View

Sets a user-visible string identifying the view’s functionality. ビューのもつ機能性を識別するユーザ可視の文字列を設定します。

func touchBarItemPresence(TouchBarItemPresence) -> View

Sets the behavior of the view when being customized by the user. ビューの挙動を設定します、ユーザによってカスタマイズされている時に。

struct TouchBar

A container for a view that you can show in the Touch Bar.

enum TouchBarItemPresence

Options that affect user customization of the Touch Bar.
