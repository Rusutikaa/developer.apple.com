Previews プレビュー

Generate dynamic, interactive previews of your custom views. あなたのあつらえのビューの動的な、相互作用するプレビューを生成します。

Framework フレームワーク

  • SwiftUI

Topics 話題

Preview Creation プレビュー作成

protocol PreviewProvider

A type that produces view previews in Xcode. ある型、それはビュープレビューをXcodeにおいて生み出すものです。

Preview Customization プレビューカスタマイゼーション

struct PreviewDevice

The simulator device that runs a preview. プレビューを走らせるシミュレータデバイス。

enum PreviewPlatform

The operating system on which to run the preview. それの上でプレビューが動作するオペレーティングシステム。

enum PreviewLayout

The size constraint for a preview. プレビューに対するサイズ制約。
