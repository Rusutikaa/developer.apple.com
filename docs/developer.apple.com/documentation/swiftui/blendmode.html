Compositing Views ビューを合成する

func blendMode(BlendMode) -> View

Sets the blend mode for compositing the view with overlapping views. 混合モードを、重なり合っているビューをもつビューの合成に対して設定します。

func compositingGroup() -> View

Wraps the view in a compositing group. ビューをある合成グループの中に包みます。

func drawingGroup(opaque: Bool, colorMode: ColorRenderingMode) -> View

Composites the view’s contents into an offscreen image before final display. ビューのもつ内容をある画面外の画像へと最終的表示の前に合成します。

enum ColorRenderingMode
