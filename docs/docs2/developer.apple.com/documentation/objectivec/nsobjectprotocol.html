NSObjectProtocol

Declaration 宣言

Overview 概要

Note 注意

This protocol is imported into Swift with the name NSObjectProtocol.

An object that conforms to this protocol can be considered a first-class object. Such an object can be asked about its:

  • Class, and the place of its class in the inheritance hierarchy.

  • Conformance to protocols.

  • Ability to respond to a particular message.

The Cocoa root class NSObject adopts this protocol, so all objects inheriting from NSObject have the features described by this protocol.

Topics 話題

Identifying Classes クラスの識別

var superclass: AnyClass?

Returns the class object for the receiver’s superclass. レシーバのスーパークラスに対するクラスオブジェクトを返します。

Required. 必須。

Identifying and Comparing Objects

func isEqual(Any?) -> Bool

Returns a Boolean value that indicates whether the receiver and a given object are equal. レシーバと与えられたオブジェクトが等しいかどうかを指し示すブール値を返します。

Required. 必須。

var hash: Int

Returns an integer that can be used as a table address in a hash table structure.

Required. 必須。

func `self`() -> Self

Returns the receiver. レシーバを返します。

Required. 必須。

Testing Object Inheritance, Behavior, and Conformance

func isKind(of: AnyClass) -> Bool

Returns a Boolean value that indicates whether the receiver is an instance of given class or an instance of any class that inherits from that class.

Required. 必須。

func isMember(of: AnyClass) -> Bool

Returns a Boolean value that indicates whether the receiver is an instance of a given class.

Required. 必須。

func responds(to: Selector!) -> Bool

Returns a Boolean value that indicates whether the receiver implements or inherits a method that can respond to a specified message.

Required. 必須。

func conforms(to: Protocol) -> Bool

Returns a Boolean value that indicates whether the receiver conforms to a given protocol. レシーバが指定したプロトコルに準拠するかどうかを指し示すブール値を返します。

Required. 必須。

Describing Objects オブジェクトの解説

var description: String

Returns a string that describes the contents of the receiver.

Required. 必須。

var debugDescription: String

Returns a string that describes the contents of the receiver for presentation in the debugger.

Sending Messages メッセージ送信

func perform(Selector!) -> Unmanaged<AnyObject>!

Sends a specified message to the receiver and returns the result of the message.

Required. 必須。

func perform(Selector!, with: Any!) -> Unmanaged<AnyObject>!

Sends a message to the receiver with an object as the argument.

Required. 必須。

func perform(Selector!, with: Any!, with: Any!) -> Unmanaged<AnyObject>!

Sends a message to the receiver with two objects as arguments.

Required. 必須。

Identifying Proxies

func isProxy() -> Bool

Returns a Boolean value that indicates whether the receiver does not descend from NSObject.

Required. 必須。

Object Basics オブジェクト基礎

class NSObject

The root class of most Objective-C class hierarchies, from which subclasses inherit a basic interface to the runtime system and the ability to behave as Objective-C objects. たいていのObjective-Cクラス階層のルートクラス、そこからサブクラスは基本的なインターフェイスとObjective-Cオブジェクトとして振舞う能力を継承します。

NSKeyValueCoding

A mechanism by which you can access the properties of an object indirectly by name or key. それによってあなたがあるオブジェクトのプロパティに名前またはキーによって間接的にアクセスできるある仕組み。
