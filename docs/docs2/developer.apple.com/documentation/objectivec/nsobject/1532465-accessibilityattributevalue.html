Instance Method インスタンスメソッド

accessibilityAttributeValue(_:)

Returns the value of the specified attribute in the receiver.

Deprecated 非推奨

This method is deprecated. Use the new method-based API instead. For more information, see Accessibility Programming Guide for OS X or NSAccessibilityProtocol. このメソッドは非推奨にされます。新しいメソッド基盤のAPIを代わりに使ってください。さらなる情報として、Accessibility Programming Guide for OS XまたはNSAccessibilityProtocolを見てください。

Declaration 宣言

Parameters パラメータ

attribute

The name of the attribute. See Constants for lists of attribute names.

Discussion 解説

User interface classes must implement this method. Subclasses should invoke the superclass's implementation, if it exists, if attribute is not implemented in the subclass.
