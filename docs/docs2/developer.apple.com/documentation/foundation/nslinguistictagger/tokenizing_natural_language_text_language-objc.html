Tokenizing Natural Language Text 自然言語テキストをトークン化する

Enumerate the words in a string. ある文字列の中の単語を列挙します。

Framework フレームワーク

  • Foundation ファウンデーション

On This Page この項には

  • Overview 概要
  • See Also 参照

Overview 概要

When you work with natural language text, it's often useful to tokenize the text into individual words. Using NSLinguisticTagger to enumerate words, rather than simply splitting components by whitespace, ensures correct behavior in multiple scripts and languages. For example, neither Chinese nor Japanese uses spaces to delimit words. あなたが自然言語テキストを扱う時、しばしば役立つのはテキストを個々の単語へとトークン化することです。NSLinguisticTaggerを単語を列挙するために使うことは、単純に構成要素を空白によって分割するよりむしろ、複数の書体と言語において正しい挙動を確かなものにします。例えば、中国語も日本語も単語を区切るために空白を使いません。

The example and accompanying steps below show how you use NSLinguisticTagger to enumerate over the words in natural language text. 下の例と付随する手順は、どのようにあなたがNSLinguisticTaggerを使って自然言語テキストの中の単語を列挙するかを示します。

  1. Create an instance of NSLinguisticTagger, specifying NSLinguisticTagSchemeTokenType as the tag scheme to be used. NSLinguisticTaggerのインスタンスを作成してください、NSLinguisticTagSchemeTokenTypeを使用タグスキームとして指定します。

  2. Set the string property of the linguistic tagger to the natural language text. その言語学的タガーのstringプロパティを自然言語テキストに設定してください。

  3. Enumerate over the entire range of the string by calling the enumerateTagsInRange:unit:scheme:options:usingBlock: method, specifying NSLinguisticTaggerUnitWord as the tag unit and NSLinguisticTagSchemeTokenType as the tag scheme to enumerate, and omitting any punctuation or whitespace. 文字列の範囲全体にわたっての列挙をenumerateTagsInRange:unit:scheme:options:usingBlock:メソッドを呼びだして、NSLinguisticTaggerUnitWordをタグ単位としてそしてNSLinguisticTagSchemeTokenTypeをタグスキームとして列挙に対して指定して、そしてあらゆる句読点や空白を省略することによって行ってください。

  4. In the enumeration block, take a substring of the original text at tokenRange to obtain each word. 列挙ブロックにおいて、元のテキストの下位文字列をtokenRangeで取得することによって各単語を得てください。

  5. Run this code to print out each word in text on a new line. このコードを実行してtextの中の各単語をひとつの新しい行にプリントしてください。
