NSFileCreationDate

The key in a file attribute dictionary whose value indicates the file's creation date. ファイル属性辞書の中のキー、それの値はファイルの作成日を指し示します。

SDKs

  • macOS 10.2+
  • Mac Catalyst 13.0+
  • tvOS 9.0+
  • watchOS 2.0+
