Type Method 型メソッド

valueWithRange:

Creates a new value object containing the specified Foundation range structure. 指定されたFoundation範囲構造体を含んでいる新しい値オブジェクトを作成します。

Parameters パラメータ

range

The value for the new object. 新しいオブジェクトに対する値。

Return Value 戻り値

rangeValue

Related Documentation 関連文書

NSRange

A structure used to describe a portion of a series, such as characters in a string or objects in an array. あるシリーズの一部分、例えば文字列の中の文字または配列の中のオブジェクトを記述するのに使われる構造体。
