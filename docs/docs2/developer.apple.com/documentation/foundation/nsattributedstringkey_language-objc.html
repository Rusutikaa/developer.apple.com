Type Alias

NSAttributedStringKey

Attributes that you can apply to text in an attributed string. あなたが属性付文字列にの中のテキストに適用できる属性。

NSAccessibilityMarkedMisspelledTextAttribute

Misspelled text that is visibly marked as misspelled (NSNumber as a Boolean value). If you’re implementing a custom text-editing app, use NSAccessibilityMarkedMisspelledTextAttribute to ensure that VoiceOver properly identifies misspelled text to users. 綴り間違いとして目に見えるように印される綴り間違いテキスト（ブール値としてのNSNumber）。あなたがあつらえのテキスト編集アプリを実装しているならば、NSAccessibilityMarkedMisspelledTextAttributeを使ってVoiceOverプロパティが綴り間違いテキストを適切に識別することをユーザのために確実にしてください。

NSAccessibilityMisspelledTextAttribute

Misspelled text that isn’t necessarily visibly marked as misspelled (NSNumber as a Boolean value). 綴り間違いとして目に見えるように印される必要がない綴り間違いテキスト（ブール値としてのNSNumber）。 Beginning in macOS 10.9, VoiceOver no longer checks for this attribute; instead, VoiceOver uses NSAccessibilityMarkedMisspelledTextAttribute.

NSAccessibilityShadowTextAttribute

Text shadow (NSNumber as a Boolean value). テキストシャドウ（ブール値としてのNSNumber）。

NSAccessibilityStrikethroughTextAttribute

Text strikethrough (NSNumber as a Boolean value). テキスト取り消し線（ブール値としてのNSNumber）。

NSAccessibilityStrikethroughColorTextAttribute

Text strikethrough color (CGColorRef). テキスト取り消し線の色（CGColorRef）。

NSAccessibilitySuperscriptTextAttribute

Text superscript style (NSNumber). Values > 0 are superscript; values < 0 are subscript. テキスト上付きスタイル（NSNumber）。値 > 0 は上付きです；値 < 0 下付きです。

NSAccessibilityUnderlineTextAttribute

Text underline style (NSNumber). テキスト下線スタイル（NSNumber）。

NSAccessibilityUnderlineColorTextAttribute

Text underline color (CGColorRef). テキスト下線色（CGColorRef）。

Type Properties 型プロパティ

UIAccessibilitySpeechAttributeSpellOut

UIAccessibilitySpeechAttributeIPANotation
UIAccessibilitySpeechAttributeLanguage

A key indicating the language to use when speaking a string. 文字列を音声で語る場合に使う言語を指し示している示すキー。

UIAccessibilitySpeechAttributePitch

A key indicating the pitch to apply to spoken content. 音声で語られる内容に適用される音調を指し示しいるキー。

UIAccessibilitySpeechAttributePunctuation

A key indicating whether punctuation is spoken. 句読点が音声で語られるかどうかを指し示しているキー。

UIAccessibilitySpeechAttributeQueueAnnouncement

A key indicating whether the spoken text is queued behind, or interrupts, existing spoken content. 音声で語られるテキストが、既存の音声で語られる内容に後ろから、または割り込んで、待ち行列に加えられるか、どうかを指し示しているキー。

UIAccessibilityTextAttributeCustom

A key for specifying custom attributes to apply to the text. テキストに適用するあつらえの属性を指定するためのキー。

UIAccessibilityTextAttributeHeadingLevel

A key for specifying the heading level of the text. テキストの見出しレベルを指定するためのキー。
