Instance Method インスタンスメソッド

allowsNaturalLanguage

Returns a Boolean value that indicates whether the receiver attempts to process dates entered as a vernacular string.

SDK

  • macOS 10.4–10.9Deprecated

Framework フレームワーク

  • Foundation ファウンデーション

On This Page この項には

  • Declaration 宣言
  • Return Value 戻り値
  • Discussion 議論
  • See Also 参照

Declaration 宣言

Return Value 戻り値

YES if the receiver attempts to process dates entered as a vernacular string (“today,” “next week,” “dinner time,” and so on), otherwise NO.

Discussion 議論

Natural-language processing supports only a limited set of colloquial phrases, primarily in English. It may give unexpected results, and its use is strongly discouraged. それは予想しない結果を与えるかもしれません、そしてそれの利用は強く反対されます。

Special Considerations 特別な注意事項

This method is for use with formatters using NSDateFormatterBehavior10_0 behavior.
