NSXMLNodeKind

NSXMLNode declares the following constants of type NSXMLNodeKind for specifying a node’s kind in the initializer methods initWithKind: and initWithKind:options::

SDK

  • macOS 10.4+

  • Foundation ファウンデーション

Topics 話題

Constants 定数

NSXMLInvalidKind

Indicates a node object created without a valid kind being specified (as returned by the kind method). 指定される有効な種類なしで作成されるノードオブジェクトを指し示します（kindメソッドによって返される通りに）。

NSXMLDocumentKind

Specifies a document node. 書類ノードを指定します。

NSXMLElementKind

Specifies an element node. 要素ノードを指定します。

NSXMLAttributeKind

Specifies an attribute node 属性ノードを指定します。

NSXMLNamespaceKind

Specifies a namespace node. 名前空間ノードを指定します。

NSXMLProcessingInstructionKind

Specifies a processing-instruction node. 処理命令ノードを指定します。

NSXMLCommentKind

Specifies a comment node. コメントノードを指定します。

NSXMLTextKind

Specifies a text node. テキストノードを指定します。

NSXMLDTDKind

Specifies a document-type declaration (DTD) node. 書類型宣言（DTD）ノードを指定します。

NSXMLEntityDeclarationKind

Specifies an entity-declaration node. 空宣言ノードを指定します。

NSXMLAttributeDeclarationKind

Specifies an attribute-list declaration node. 属性リスト宣言ノードを指定します。

NSXMLElementDeclarationKind

Specifies an element declaration node. 要素宣言ノードを指定します。

NSXMLNotationDeclarationKind

Specifies a notation declaration node. 表記法宣言ノードを指定します。
