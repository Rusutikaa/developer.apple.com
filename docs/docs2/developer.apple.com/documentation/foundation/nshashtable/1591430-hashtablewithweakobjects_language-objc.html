Type Method 型メソッド

hashTableWithWeakObjects

Returns a new hash table for storing weak references to its contents. それの内容への弱い参照を格納するため新しいハッシュテーブルを返します。

Deprecated 非推奨

Use weakObjectsHashTable instead.

SDK

  • macOS 10.5–10.8Deprecated

Framework フレームワーク

  • Foundation ファウンデーション

  • Declaration 宣言

Declaration 宣言

Return Value 戻り値

A new has table that uses the options NSHashTableZeroingWeakMemory and NSPointerFunctionsObjectPersonality and has an initial capacity of 0.

Discussion 議論

This method is not supported under Automatic Reference Counting (ARC).
