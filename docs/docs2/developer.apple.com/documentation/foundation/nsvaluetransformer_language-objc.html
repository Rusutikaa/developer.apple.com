Class

NSValueTransformer

An abstract class used to transform values from one representation to another. 一方の表現から他方へ値を変換するのに使われる抽象クラス。

Declaration 宣言

Overview 概要

You create a value transformer by subclassing NSValueTransformer and overriding the necessary methods to provide the required custom transformation. You then register the value transformer using the setValueTransformer:forName: method, so that other parts of your app can access it by name with valueTransformerForName:.

Use the transformedValue: method to transform a value from one representation into another. If a value transformer designates that its transformation is reversible by returning YES for allowsReverseTransformation, you can also use the reverseTransformedValue: to perform the transformation in reverse. For example, reversing the characters in a string is a reversible operation, whereas changing the characters in a string to be uppercase is a nonreversible operation. たとえば、ある文字列中の文字を逆にすることは元に戻せる操作です、その一方である文字列の中の文字をアップケースであるように変えることは元に戻せない操作です。

A value transformer can take inputs of one type and return a value of a different type. For example, a value transformer could take an NSImage or UIImage object and return an NSData object containing the PNG representation of that image. ある値変換器は、１つの型の入力を取り、そして異なる型の値を返すことが出来ます。たとえば、ある値変換器は、NSImageまたはUIImageオブジェクトを取り、そしてその画像のPNG表現を含んでいるNSDataオブジェクトを返せます。

Topics 話題

Using the Name-Based Registry 名前基盤のレジストリ（登録簿）を使う

+ setValueTransformer:forName:

Registers the provided value transformer with a given identifier. 提供された値変換子をある与えられた識別子で登録します。

+ valueTransformerForName:

Returns the value transformer identified by a given identifier. 与えられた識別子によって識別される値変換子を返します。

+ valueTransformerNames

Returns an array of all the registered value transformers. 全ての登録された値変換子からなる配列を返します。

NSValueTransformerName

Named value transformers defined by NSValueTransformer. NSValueTransformerによって定義される名前付値変換子。

Getting Information About a Transformer 変換器についての情報を取得する

+ allowsReverseTransformation

Returns a Boolean value that indicates whether the receiver can reverse a transformation. あるブール値を返します、それはレシーバがある変換を逆にできるかどうかを指し示します。

+ transformedValueClass

Returns the class of the value returned by the receiver for a forward transformation. レシーバによって返される値のクラスを前方変換のために返します。

Transforming Values 値を変換する

- transformedValue:

Returns the result of transforming a given value. 与えられた値の変換の結果を返します。

- reverseTransformedValue:

Returns the result of the reverse transformation of a given value. 与えられた値の逆変換の結果を返します。
