Type Method 型メソッド

predicateWithLeftExpression:rightExpression:modifier:type:options:

Creates and returns a predicate of a given type formed by combining given left and right expressions using a given modifier and options.

SDKs

  • iOS 3.0+
  • macOS 10.4+
  • Mac Catalyst 13.0+
  • tvOS 9.0+
  • watchOS 2.0+

Framework フレームワーク

  • Foundation ファウンデーション

On This Page この項には

  • Declaration 宣言
  • Parameters パラメータ
  • Return Value 戻り値
  • See Also 参照

Declaration 宣言

Parameters パラメータ

lhs

The left hand expression. 左手の式。

rhs

The right hand expression. 右手の式。

modifier

The modifier to apply. 適用する修正子。

type

The predicate operator type. 述部演算子型。

options オプション

The options to apply (see NSComparisonPredicateOptions). For no options, pass 0. オプション無しに対しては、0を渡してください。

Return Value 戻り値

A new predicate of type type formed by combining the given left and right expressions using the modifier and options.
