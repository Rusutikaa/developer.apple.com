NSJSONSerialization

An object that converts between JSON and the equivalent Foundation objects. JSONと同等のFoundationオブジェクト間で変換するオブジェクト。

Declaration 宣言

Overview 概要

You use the NSJSONSerialization class to convert JSON to Foundation objects and convert Foundation objects to JSON. あなたはNSJSONSerializationオブジェクトを使って、JSONをFoundationオブジェクトへと変換したり、FoundationオブジェクトをJSONへと変換します。

A Foundation object that may be converted to JSON must have the following properties: JSONに変換できるかもしれないFoundationオブジェクトは、以下の性質を持たなければなりません：

  • The top level object is an NSArray or NSDictionary. トップレベルオブジェクトは、NSArrayまたはNSDictionaryである。

  • All objects are instances of NSString, NSNumber, NSArray, NSDictionary, or NSNull. すべてのオブジェクトは、NSString、NSNumber、NSArray、NSDictionary、またはNSNullのインスタンスである。

  • All dictionary keys are instances of NSString. すべての辞書キーは、NSStringのインスタンスである。

  • Numbers are not NaN or infinity. 数値は、NaNまたはinfinityでない。

Other rules may apply. 他の規則が適用されるかもしれません。 Calling isValidJSONObject: or attempting a conversion are the definitive ways to tell if a given object can be converted to JSON data.

Thread Safety スレッド安全

On iOS 7 and later and macOS 10.9 and later NSJSONSerialization is thread safe.

Topics 話題

Creating a JSON Object JSONオブジェクトを作成する

+ JSONObjectWithData:options:error:

Returns a Foundation object from given JSON data. 与えられたJSONデータからFoundationオブジェクトを返します。

+ JSONObjectWithStream:options:error:

Returns a Foundation object from JSON data in a given stream. 与えられたストリームの中のJSONデータからFoundationオブジェクトを返します。

Creating JSON Data JSONデータを作成する

+ dataWithJSONObject:options:error:

Returns JSON data from a Foundation object. FoundationオブジェクトからJSONデータを返します。

+ writeJSONObject:toStream:options:error:

Writes a given JSON object to a stream. 与えられたJSONオブジェクトをストリームに書き込みます。

+ isValidJSONObject:

Returns a Boolean value that indicates whether a given object can be converted to JSON data. 与えられたオブジェクトがJSONデータに変換できるかどうかを示すブール値を返します。

Constants 定数

NSJSONReadingOptions

Options used when creating Foundation objects from JSON data—see JSONObjectWithData:options:error: and JSONObjectWithStream:options:error:.

NSJSONWritingOptions

Options for writing JSON data. JSONデータの書き込みに対するオプション。
