Directory Locations ディレクトリ位置

NSApplicationDirectory

Supported applications (/Applications). サポートされるアプリケーション（/Applications）。

NSDemoApplicationDirectory

Unsupported applications and demonstration versions. サポートされないアプリケーションおよびデモンストレーション版。

NSDeveloperApplicationDirectory

Developer applications (/Developer/Applications). 開発アプリケーション（/Developer/Applications）。

NSAdminApplicationDirectory

System and network administration applications. システムおよびネットワーク管理アプリケーション。

NSLibraryDirectory

Various user-visible documentation, support, and configuration files (/Library). 様々なユーザが見ることができる書類、サポート、そして構成設定ファイル（/Library）。

NSDeveloperDirectory

Developer resources (/Developer). 開発リソース（/Developer）。

NSUserDirectory

User home directories (/Users). ユーザホームディレクトリ（/Users）。

NSDocumentationDirectory

Documentation. 書類。

NSDocumentDirectory

Document directory. 書類ディレクトリ。

NSCoreServiceDirectory

Core services (System/Library/CoreServices). Coreサービス（System/Library/CoreServices）。

NSAutosavedInformationDirectory

The user’s autosaved documents (Library/Autosave Information). ユーザの自動保存書類（Library/Autosave Information）。

NSDesktopDirectory

The user’s desktop directory. ユーザのデスクトップディレクトリ。

NSCachesDirectory

Discardable cache files (Library/Caches). 放棄可能なキャッシュファイル（Library/Caches）。

NSApplicationSupportDirectory

Application support files (Library/Application Support). アプリケーションサポートファイル（Library/Application Support）。

NSDownloadsDirectory

The user’s downloads directory. ユーザのダウンロードディレクトリ。

NSInputMethodsDirectory

Input Methods (Library/Input Methods). Input Methods (Library/Input Methods)。

NSMoviesDirectory

The user’s Movies directory (~/Movies). ユーザの ディレクトリ（~/Movies）。

NSMusicDirectory

The user’s Music directory (~/Music). ユーザのMusicディレクトリ（~/Music）。

NSPicturesDirectory

The user’s Pictures directory (~/Pictures). ユーザのPicturesディレクトリ（~/Pictures）。

NSPrinterDescriptionDirectory

The system’s PPDs directory (Library/Printers/PPDs). システムのもつPPDディレクトリ（Library/Printers/PPDs）。

NSPreferencePanesDirectory

The PreferencePanes directory for use with System Preferences (Library/PreferencePanes). システム環境設定で使うためのPreferencePanesディレクトリ（Library/PreferencePanes）。

NSApplicationScriptsDirectory

The user scripts folder for the calling application (~/Library/Application Scripts/<code-signing-id>. アプリケーションを呼び出すためのユーザスクリプトフォルダ（~/Library/Application Scripts/<code-signing-id>）。

NSItemReplacementDirectory

The constant used to create a temporary directory. 一時ディレクトリを作成するのに使われる定数。

NSAllApplicationsDirectory

All directories where applications can be stored. アプリケーションが格納されることができる全てのディレクトリ。

NSAllLibrariesDirectory

All directories where resources can be stored. リソースが格納されることができる全てのディレクトリ。

NSTrashDirectory

The trash directory. ゴミ箱ディレクトリ。
