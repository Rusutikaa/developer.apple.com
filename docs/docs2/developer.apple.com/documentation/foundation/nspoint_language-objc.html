Type Alias

NSPoint

A point in a Cartesian coordinate system. デカルト座標システムの中のある点。

SDKs

  • macOS 10.0+
  • Mac Catalyst 13.0+

Framework フレームワーク

  • Foundation ファウンデーション

  • Declaration 宣言
  • Discussion 議論
  • Topics 話題
  • See Also 参照

Declaration 宣言

Discussion 議論

Prior to OS X v10.5 the coordinates were represented by float values rather than CGFloat values. OS X v10.5より前は座標はfloat値によって表されました、CGFloat値ではなく。

When building for 64 bit systems, or building 32 bit like 64 bit, NSPoint is typedef’d to CGPoint. 64ビットシステムに対してビルドする、または64ビットのように32ビットをビルドする場合、NSPointはCGPointにtypedefされます。

Topics 話題

Managing Points ポイントの管理

NSEqualPoints

Returns a Boolean value that indicates whether two points are equal. ２つの地点が等しいかどうかを指し示すブール値を返します。

NSMakePoint

Creates a new NSPoint from the specified values. 新しいNSPointを指定された値から作成します。

NSPointFromString

Returns a point from a text-based representation. テキストに基づく表現から、ある点を返します。

NSStringFromPoint

Returns a string representation of a point. ある点の文字列表現を返します。

NSPointFromCGPoint

Returns an NSPoint typecast from a CGPoint. CGPointから型キャストされたNSPointを返します。

NSPointToCGPoint

Returns a CGPoint typecast from an NSPoint. NSPointから型キャストされた、CGPointを返します。

Zero Constant ゼロ定数

NSZeroPoint

An NSPoint structure with both x and y coordinates set to 0. あるNSPoint構造体で、xとy座標の両方とも0に設定されます。

Related Types 関連する型

NSPointArray

Type indicating a parameter is array of NSPoint structures. パラメータを指し示している型は、NSPoint構造体の配列です。

NSPointPointer

Type indicating a parameter is a pointer to an NSPoint structure. パラメータを指し示している型は、NSPoint構造体へのポインタです。
