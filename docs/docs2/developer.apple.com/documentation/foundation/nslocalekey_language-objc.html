Type Alias

NSLocaleKey

The keys used to access components of a locale. あるロケールの構成要素にアクセスするために使われるキー。

SDKs

  • iOS 10.0+
  • macOS 10.12+
  • Mac Catalyst 13.0+
  • tvOS 10.0+
  • watchOS 3.0+

Framework フレームワーク

  • Foundation ファウンデーション

Declaration 宣言

Discussion 議論

Use these keys with the methods objectForKey: and displayNameForKey:value:.

Topics 話題

Keys キー

NSLocaleIdentifier

The locale identifier. ロケール識別子。

NSLocaleCountryCode

The locale country code. ロケール国名コード。

NSLocaleLanguageCode

The locale language code. ロケール言語コード。

NSLocaleScriptCode

The locale script code. ロケール書体コード。

NSLocaleVariantCode

The locale variant code. ロケール変種コード。

NSLocaleExemplarCharacterSet

The exemplar character set for the locale. このロケールに対する典型文字セット。

NSLocaleCalendar

The calendar associated with the locale. このロケールと結び付けられたカレンダー。

NSLocaleCollationIdentifier

The collation associated with the locale. ロケールと結び付けられた照合。

NSLocaleCollatorIdentifier

The collation identifier for the locale. このロケールに対する照合識別子。

NSLocaleUsesMetricSystem

A flag that indicates whether the locale uses the metric system. そのロケールがメートル法を使うかどうかを指し示すフラグ。

NSLocaleMeasurementSystem

The measurement system associated with the locale. このロケールと結び付けられた単位系。

NSLocaleDecimalSeparator

The decimal separator associated with the locale. このロケールと結び付けられた小数分離子。

NSLocaleGroupingSeparator

The numeric grouping separator associated with the locale. このロケールと結び付けられた数値グループ分離子。

NSLocaleCurrencySymbol

The currency symbol associated with the locale. このロケールと結び付けられた通貨表象。

NSLocaleCurrencyCode

The currency code associated with the locale. このロケールと結び付けられた通貨コード。

NSLocaleQuotationEndDelimiterKey

The end quotation symbol associated with the locale. このロケールと結び付けられた終了引用表象。

NSLocaleQuotationBeginDelimiterKey

The begin quotation symbol associated with the locale. このロケールと結び付けられた開始引用表象。

NSLocaleAlternateQuotationEndDelimiterKey

The alternate end quotation symbol associated with the locale. このロケールと結び付けられた代替終了引用表象。

NSLocaleAlternateQuotationBeginDelimiterKey

The alternating begin quotation symbol associated with the locale. このロケールと結び付けられた代替開始引用表象。

See Also 参照

Accessing Locale Information by Key キーによってロケール情報にアクセスする

- objectForKey:

Returns the value of the component corresponding to the specified key. 指定されたキーに対応する構成要素の値を返します。

- displayNameForKey:value:

Returns the display name for the given locale component value. 与えられたロケール構成要素値に対する表示名を返します。
