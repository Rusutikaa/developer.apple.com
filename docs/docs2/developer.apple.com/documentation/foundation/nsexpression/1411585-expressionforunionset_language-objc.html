Type Method 型メソッド

expressionForUnionSet:with:

Returns a new NSExpression object that represent the union of a given set and collection. 新しいNSExpressionオブジェクトを返します、それは与えられた集合とコレクションの和集合を表します。

SDKs

  • iOS 3.0+
  • macOS 10.5+
  • Mac Catalyst 13.0+
  • tvOS 9.0+
  • watchOS 2.0+

Framework フレームワーク

  • Foundation ファウンデーション

On This Page この項には

  • Declaration 宣言
  • Parameters パラメータ
  • Return Value 戻り値
  • See Also 参照

Declaration 宣言

Parameters パラメータ

left

An expression that evaluates to an NSSet object. ある式、それはひとつのNSSetオブジェクトに評価されるものです。

right

An expression that evaluates to a collection object (an instance of NSArray, NSSet, or NSDictionary). ある式、それはひとつのコレクションオブジェクトに評価されるものです（NSArray、NSSet、またはNSDictionaryのインスタンス）。

Return Value 戻り値

An new NSExpression object that represents the union of left and right. 新しいNSExpressionオブジェクト、それはleftとrightの和集合です。

See Also 参照

Creating a Collection Expression コレクション式を作成する

+ expressionForAggregate:

Returns a new aggregate expression for a given collection. 与えられたコレクションに対する新しい集計式を返します。

+ expressionForIntersectSet:with:

Returns a new NSExpression object that represent the intersection of a given set and collection. 新しいNSExpressionオブジェクトを返します、それは与えられた集合とコレクションの交わりを表します。

+ expressionForMinusSet:with:

Returns a new NSExpression object that represent the subtraction of a given collection from a given set. 新しいNSExpressionオブジェクトを返します、それは与えられたコレクションを与えられた集合から引いたものを表します。
