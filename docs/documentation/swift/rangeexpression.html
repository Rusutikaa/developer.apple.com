Protocol

RangeExpression

A type which can be used to slice a collection. A RangeExpression can convert itself to a Range<Bound> of indices within a given collection; the collection can then slice itself with that Range.
コレクションをスライスするために使われるある型。RangeExpressionは、それ自身を与えられたコレクション内のインデックスのRange<Bound>に変換できます；そのコレクションはそれからそれ自身そのRangeでスライスできます。

struct PartialRangeUpTo

A partial half-open interval up to, but not including, an upper bound.
上側の境界までの、しかしそれを含まない部分的半開間隔。

struct PartialRangeThrough

A partial half-open interval up to, and including, an upper bound.
ある上側の境界までの、そしてそれを含んでいる部分的な半開間隔。

struct PartialRangeFrom

A partial interval extending upward from a lower bound.
ある下側の境界から上向きに拡張する部分的な間隔。

struct CountablePartialRangeFrom

A partial interval extending upward from a lower bound that forms a sequence of increasing values.
ある下側の境界から上向きに拡張する部分的な間隔、それは漸増するいくらかの値からなるシーケンスを形成します。
