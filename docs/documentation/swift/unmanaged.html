Unmanaged

A type for propagating an unmanaged object reference.
ある管理されないオブジェクト参照を伝達するための型。

SDK

  • Xcode 6.0.1+

Overview
概要

When you use this type, you become partially responsible for keeping the object alive.
あなたがこの型を使う場合、あなたはオブジェクト存続を保っていることに対して部分的に責任を負うようになります。

Topics
話題

Instance Methods
インスタンスメソッド

func autorelease()

Performs an unbalanced autorelease of the object.
オブジェクトのある不均衡なオートリリース（自動解放）を実行します。

func release()

Performs an unbalanced release of the object.
オブジェクトのある不均衡なリリース（解放）を実行します。

func retain()

Performs an unbalanced retain of the object.
オブジェクトのある不均衡なリテイン（保持）を実行します。

func takeRetainedValue()

Gets the value of this unmanaged reference as a managed reference and consumes an unbalanced retain of it.
この管理されない参照の値を管理された参照として取得します、そしてそれのある不均衡なリテイン（保持）を使い果たします。

func takeUnretainedValue()

Gets the value of this unmanaged reference as a managed reference without consuming an unbalanced retain of it.
この管理されない参照の値を管理された参照として取得します、それのある不均衡なリテイン（保持）を使い果たすことなしにです。

func toOpaque()

Unsafely converts an unmanaged class reference to a pointer.
管理されないクラス参照をポインタへと安全でない変換をする。

Type Methods
型メソッド

static func fromOpaque(UnsafeRawPointer)

Unsafely turns an opaque C pointer into an unmanaged class reference.
不透明Cポインタの管理されないクラス参照への安全でない転換。

static func passRetained(Instance)

Creates an unmanaged reference with an unbalanced retain.
管理されない参照を不均衡なリテイン（保持）で作成します。

static func passUnretained(Instance)

Creates an unmanaged reference without performing an unbalanced retain.
管理されない参照を不均衡なリテイン（保持）を実行することなしに作成します。
