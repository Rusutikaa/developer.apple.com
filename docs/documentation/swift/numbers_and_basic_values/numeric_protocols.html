Basic Arithmetic
基本算術

protocol Numeric

Declares methods backing binary arithmetic operators–such as +, - and *–and their mutating counterparts.
バイナリ算術演算 – 例えば+、-そして* – そしてそれらの可変の相当物などを裏打ちするメソッドを宣言します。

Beta
protocol SignedNumeric

A type that can represent both positive and negative values.
正と負の値の両方を表せる型。

Beta
protocol Strideable

Conforming types are notionally continuous, one-dimensional values that can be offset and measured.
準拠する型は、概念上連続的な、一次元の値で、それはオフセットできて計測できるものです。

Integer
整数

protocol BinaryInteger

An integer type with a binary representation.
バイナリ表現での整数型。

Beta
protocol FixedWidthInteger

An integer type that uses a fixed size for every instance.
あらゆるインスタンスに対して固定された大きさを使う整数型。

Beta
protocol SignedInteger

An integer type that can represent both positive and negative values.
正および負の値の両方を表すことができる整数型。

protocol UnsignedInteger

An integer type that can represent only nonnegative values.
非負値だけを表すことができる整数型。

Floating Point
浮動小数点

protocol FloatingPoint

A floating-point numeric type.
浮動小数点数値型。

protocol BinaryFloatingPoint

A radix-2 (binary) floating-point type.
あるradix-2（バイナリ）浮動小数点型。

Deprecated
非推奨

protocol _Incrementable

This protocol is an implementation detail of Integer; do not use it directly.
このプロトコルはIntegerの実装詳細です；それを直接に使わないでください。
