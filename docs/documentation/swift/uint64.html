UInt64

An unsigned integer value type.
符号なし整数値型。

Overview
概要

On 32-bit platforms, UInt64 is the same size as UInt6432, and on 64-bit platforms, UInt64 is the same size as UInt6464.
32ビットプラットホームでは、UInt64はUInt6432と同じ大きさです、そして64ビットプラットホームでは、UInt64はUInt6464と同じ大きさです。
