UInt

Overview
概要

On 32-bit platforms, UInt is the same size as UInt32, and on 64-bit platforms, UInt is the same size as UInt64.
32ビットプラットホーム上では、UIntはUInt32と同じ大きさです、そして64ビットプラットホーム上では、UIntはUInt64と同じ大きさです。
