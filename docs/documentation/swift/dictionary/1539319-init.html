Initializer

init(minimumCapacity:)

Creates a dictionary with at least the given number of elements worth of buffer.
少なくとも指定された要素の数に相当するバッファを持つ辞書を作成します。

SDK

  • Xcode 6.0.1+

Framework
フレームワーク

Declaration
宣言

Parameters
パラメータ

minimumCapacity

The minimum number of key-value pairs to allocate buffer for in the new dictionary.
新しい辞書におけるキー値ペアの最小限の数、それに対してバッファが割り当てられることになります。

Discussion
解説

Use this initializer to avoid intermediate reallocations when you know how many key-value pairs you are adding to a dictionary. The actual capacity of the created dictionary is the smallest power of 2 that is greater than or equal to minimumCapacity.
どのくらい多くのキー値ペアをあなたが辞書に加えることになるかあなたが知っている場合は、このイニシャライザを使って幾度もの中間的な再割り当てを回避してください。作成された辞書の実際の容量は、minimumCapacityより大きいか等しい最も小さい２の冪です。

Creating a Dictionary
辞書の作成

init()

Creates an empty dictionary.
空の辞書を作成します。

init<S>(uniqueKeysWithValues: S)

Creates a new dictionary from the key-value pairs in the given sequence.
新しい辞書を与えられたシーケンスの中のキー値ペアから作成します。

Beta
init<S>(S, uniquingKeysWith: (Value, Value) -> Value)

Creates a new dictionary from the key-value pairs in the given sequence, using a combining closure to determine the value for any duplicate keys.
新しい辞書を与えられたシーケンスの中のキー値ペアから作成します、結合用のクロージャを使って何らかの重複キーに対して値を決定します。

Beta
init<S>(grouping: S, by: (S.Element) -> Key)

Creates a new dictionary whose keys are the groupings returned by the given closure and whose values are arrays of the elements that returned each key.
新しい辞書を作成します、それのキーは与えられたクロージャによって返されるグループ分けです、そしてそれの値はいくらかの要素からなる配列で、それが各キーで返されるものです。
