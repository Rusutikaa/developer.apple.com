PlaygroundQuickLook

The sum of types that can be used as a Quick Look representation.
「クイックルック」表現として使われることのできる型の総和。

Overview
概要

Note
注意

PlaygroundQuickLook is deprecated, and will be removed from the standard library in a future Swift release. Customizing display for in a playground is now done using the CustomPlaygroundDisplayConvertible protocol, which does not use the PlaygroundQuickLook enum. Please remove your uses of PlaygroundQuickLook, or conditionalize your use such that it is only present when compiling with Swift 4.0 or Swift 3.2 or earlier:
PlaygroundQuickLookは推奨されません、将来のSwiftリリースでの標準ライブラリから取り除かれるでしょう。あるプレイグラウンドにおける表示のカスタマイズは、現在CustomPlaygroundDisplayConvertibleプロトコルを使って行われます、それはPlaygroundQuickLook列挙を使いません。どうかあなたのPlaygroundQuickLookの利用を取り除いてください、または例えばそれがSwift 4.0またはSwift 3.2以前でコンパイルする時にだけ存在するように、あなたの利用を条件付けてください。

#if !(swift(>=4.1) || swift(>=3.3) && !swift(>=4.0)) /* OK to use PlaygroundQuickLook */ #endif
