zip(_:_:)

Creates a sequence of pairs built out of two underlying sequences.
２つの根底にあるシーケンスから組み立てられるペアからなるシーケンスを作成します。

SDK

  • Xcode 8.0+

Parameters
パラメータ

sequence1

The first sequence or collection to zip.
組み合わされる第１のシーケンスまたはコレクション。

sequence2

The second sequence or collection to zip.
組み合わされる第２のシーケンスまたはコレクション。

Return Value
戻り値

A sequence of tuple pairs, where the elements of each pair are corresponding elements of sequence1 and sequence2.
対のタプルからなるシーケンス、そこにおいて各対の要素らは、sequence1とsequence2の要素に相当します。
